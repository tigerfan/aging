# P12 呼吸雕塑仪——呼吸训练生物反馈设备

---

## A. 需求分析文档

### 1. 产品概述与定位

**呼吸雕塑仪**是一款专为老年慢性阻塞性肺疾病（COPD）、肺康复及呼吸功能下降人群设计的智能呼吸训练设备。通过高精度气流传感器实时测量呼吸参数，结合心率变异性（HRV）监测，提供个性化的呼吸模式训练（腹式呼吸、缩唇呼吸、4-7-8呼吸法），并以3D可视化"呼吸雕塑"作为生物反馈媒介，将枯燥的呼吸训练变成艺术创作体验，显著提升老年患者的康复依从性。

**蓝海点**：肺康复（Pulmonary Rehabilitation）是COPD管理最有效的非药物干预，但居家化设备几乎空白；呼吸训练设备（吹气瓶）极其原始，完全没有智能化。

### 2. 目标用户画像

| 维度 | 描述 |
|------|------|
| 核心用户 | 60–80岁，COPD稳定期患者（GOLD I–III期），有呼吸功能下降的老年人 |
| 次要用户 | 肺炎/新冠后遗症康复患者，有焦虑障碍（呼吸训练为辅助干预） |
| 使用场景 | 每日2次，每次15–20分钟，坐姿，居家或养老院 |
| 医疗背景 | 呼吸科/全科医生开具康复建议，护士/治疗师随访 |
| 操作门槛 | 低，仅需按照设备语音引导呼吸 |

### 3. 市场痛点与机会

- 中国COPD患者约1亿，是老龄化死亡率最高的慢性病之一
- 居家肺康复严重不足，传统"吹乒乓球"训练无法量化效果
- 呼吸训练设备市场现有产品（Threshold IMT等）仅提供机械阻力，无数字化功能
- HRV作为自主神经功能指标，与呼吸训练效果高度相关，但鲜有设备整合
- "呼吸可视化"的艺术疗愈概念符合老年人对美好体验的追求

### 4. 功能性需求（FR）

**FR-01 呼吸参数实时采集**
- 气流传感器（差压式）测量：潮气量（TV）、呼吸频率（RR）、吸/呼比（I:E ratio）、呼气峰流速（PEF）
- 心率传感器（PPG，腕带或夹指式）同步采集心率和HRV指标（RMSSD）
- 血氧饱和度（SpO2）实时监测（训练安全保障）

**FR-02 呼吸模式训练**
- 内置训练模式：
  - 腹式呼吸（腹部扩张引导，4秒吸/4秒呼）
  - 缩唇呼吸（呼气阻力训练，2秒吸/4秒呼，COPD必备）
  - 4-7-8呼吸法（放松焦虑，4秒吸/7秒屏/8秒呼）
  - 横膈膜强化（吸气肌训练，可调阻力0–20cmH₂O）
- 训练节拍：视觉+听觉+触觉（设备体表振动）三重引导

**FR-03 3D呼吸雕塑生物反馈**
- 手机/平板屏幕实时显示3D雕塑形态（水晶球/花朵/抽象形体）
- 雕塑随呼吸参数变化：
  - 正确吸气时：雕塑向外展开（蓝色）
  - 正确呼气时：雕塑向内收缩（金色）
  - 呼吸过快/过慢：雕塑颜色变红色预警
  - 呼吸参数达标：雕塑形态完美，生成可分享图片
- 每次训练结束后，保存本次"呼吸作品"（独特的3D造型）

**FR-04 吸气肌训练（IMT）**
- 设备内置可调阻力阀（类似Threshold IMT），数字化控制阻力
- 阻力范围：0–30cmH₂O，步进1cmH₂O
- 根据最大吸气压力（MIP）自动设置训练阻力（≥30%MIP）
- 每次IMT训练自动记录吸气峰流速，评估MIP改善趋势

**FR-05 数据管理与随访**
- 每次训练：呼吸参数、HRV、SpO2、完成质量评分
- 月度报告：肺功能改善趋势图，与同类COPD患者基准对比
- 医生端：数据导出（HL7 FHIR格式），支持远程随访

**FR-06 安全监测**
- SpO2<88%持续超过1分钟：立即停止训练，提示就医
- 训练中呼吸频率>30次/分钟（呼吸窘迫）：暂停训练，通知家属
- 每次训练后：过度换气风险评估

### 5. 非功能性需求（NFR）

| 维度 | 指标 |
|------|------|
| 气流测量精度 | 潮气量误差<5%，峰流速误差<8% |
| SpO2精度 | ±2%（与医用脉氧仪对比） |
| HRV测量 | RMSSD误差<5ms（ECG对比） |
| 3D渲染帧率 | 手机端≥60fps（流畅视觉反馈） |
| 呼吸引导响应 | 视觉/听觉引导延迟<50ms |
| 清洁 | 咬嘴/接口可拆卸，煮沸消毒，每人专属咬嘴 |
| 续航 | 内置电池≥3小时连续训练，USB-C充电 |

### 6. 约束条件与合规要求

- IMT功能须作为**II类医疗器械**（吸气肌训练器）申报NMPA
- SpO2监测：符合YY 0784脉搏血氧仪标准
- 咬嘴材料：医用级硅胶，BPA-Free，通过ISO 10993生物相容性认证
- 训练禁忌声明：急性COPD加重期、自发性气胸禁用
- 云端数据合规：健康数据不用于广告，可申请完整删除

## B. 详细设计方案

### 1. 系统架构

```
[呼吸雕塑仪主机]
     ├── 差压气流传感器（潮气量/流速）
     ├── 可调阻力阀（数字化控制）
     ├── PPG传感器（心率/HRV）
     ├── SpO2传感器（红外+红光）
     ├── 触觉振动器（呼吸节拍引导）
     └── 主控MCU + BLE
          │ BLE 5.0
          ▼
[手机/平板APP]
     ├── 3D呼吸雕塑渲染（Unity）
     ├── 训练引导界面
     ├── 数据报告
     └── 医生远程连接
          ▼
[云端服务]
     ├── 肺功能基准数据库
     ├── 个性化训练处方引擎
     └── 医生随访平台
```

### 2. 硬件设计方案

**2.1 气流传感模块**
- 差压传感器：Honeywell MPRLS（范围0–250Pa，精度±0.25%），配合标准直径咬嘴
- 气流通道：直径22mm标准医用呼吸通道，与咬嘴/鼻夹配件兼容
- 温度补偿：内置温度传感器，对气体密度进行实时校正

**2.2 可调阻力系统**
- 阻力机构：数字舵机控制蝶阀开度（角度→阻力曲线标定）
- 阻力范围：0–30cmH₂O（对应蝶阀0°–60°开度）
- 实时监测：阻力+流速联合计算吸气功

**2.3 生理信号传感**
- PPG：指夹式传感器（用户选择），OSRAM SFH 7072双波长
- SpO2：与PPG集成，符合YY 0784标准
- 腹部呼吸检测（可选）：弹性腰带+应变计，检测腹部扩张（辅助腹式呼吸引导）

**2.4 主机结构**
- 外观：手持式设计，类似精致的呼吸管道雕塑感，白色哑光陶瓷涂层
- 尺寸：Φ45mm×180mm，重量<150g（单手持握舒适）
- 咬嘴：标准医用规格，磁吸快拆，独立密封包装（单人使用）
- 触觉振动：LRA线性马达（安装于握持区，呼吸节拍同步振动）

### 3. 软件设计方案

**3.1 主机固件**
```
├── 气流信号处理
│     ├── 积分法计算潮气量（实时积分流速信号）
│     ├── 峰值检测（吸气峰流速PIF，呼气峰流速PEF）
│     ├── 吸/呼比实时计算
│     └── 呼吸频率检测（过零检测法）
├── PPG信号处理（心率+HRV+SpO2）
├── 阻力控制（舵机PWM输出，闭环力控）
├── 触觉引导器（节拍发生器，与训练模式同步）
└── BLE数据服务（100ms推送一次所有传感器数据）
```

**3.2 3D雕塑渲染（Unity + Shader Graph）**
```
输入：实时呼吸参数流（潮气量、流速、I:E比）
    ↓
雕塑形变引擎：
    TV_normalized = TV / target_TV  # 归一化潮气量
    shape_scale = lerp(min_scale, max_scale, TV_normalized)
    
    # 吸气：外展，呼气：收缩
    if is_inhale:
        mesh.scale = lerp(current, max_scale, flow_ratio * dt)
        color = lerp(blue_calm, gold_bright, TV_normalized)
    else:
        mesh.scale = lerp(current, min_scale, flow_ratio * dt)
    
    # 偏差检测：频率过快时变红色
    if RR > target_RR * 1.3:
        overlay_color = warning_red
```

**3.3 医生随访平台**
- Web端（Django + REST API）
- 功能：批量患者管理、训练数据查看、远程调整阻力处方、随访记录

### 4. 核心算法设计

**4.1 个性化MIP评估与IMT处方**
```
MIP测试流程：
1. 引导用户呼气至残气量（RV）
2. 要求最大努力吸气持续3秒
3. 记录吸气峰值压力（cmH₂O）
4. 重复3次，取最大值为MIP

IMT训练处方：
- 初始阻力 = 30% MIP
- 每2周重测MIP，阻力随MIP改善同步提升
- 目标：达到正常值预测值的80%以上
```

**4.2 HRV引导的最优呼吸频率（共振频率呼吸）**
- 研究证实：每人存在一个"共振频率"（约5–7次/分钟），此频率下HRV最大化，自主神经功能最优
- 算法：在5–8次/分钟范围内逐步测试，找到RMSSD峰值对应的呼吸频率
- 训练处方：将共振频率作为基础呼吸节拍，个性化程度极高

**4.3 训练质量评分**
```
质量维度（每次训练）：
1. 频率准确度：实际RR与目标RR的吻合度（30%权重）
2. 潮气量达标率：TV>目标TV 0.8×的呼吸次数占比（30%权重）
3. 吸/呼比：I:E比与目标比例偏差（20%权重）
4. 完成时长：完成目标训练时长的百分比（20%权重）

综合评分 = 加权平均 × 100
```

### 5. 用户交互设计

**5.1 训练启动流程**
```
佩戴PPG指夹 → 拿起主机咬住咬嘴
→ APP自动识别连接（BLE）→ 显示"信号检测中..."
→ 显示当前SpO2+心率，确认安全
→ 选择今日训练模式（推荐模式自动高亮）
→ "请跟随雕塑呼吸节奏，我们开始..."
→ 3D雕塑出现，随呼吸节律膨胀收缩
→ 训练完成：展示"今日作品"（本次训练生成的独特3D雕塑造型）
→ 可分享"今日呼吸作品"到家庭群
```

**5.2 视觉引导设计**
- 雕塑底部：呼吸进度条（绿色弧形，随吸气/呼气阶段动态变化）
- 右上角：SpO2实时数值（大字体）
- 训练完成：雕塑旋转动画+粒子效果+"完美！今日呼吸训练达标"

**5.3 情感化设计细节**
- 每次成功完成完整训练：生成一个命名的"呼吸作品"（如"2026年春分的宁静"）
- 月度：生成30天呼吸作品拼贴画，可打印为装饰品
- 家庭分享：子女可收到"妈妈今日的呼吸作品"推送

### 6. 商业化路径与技术风险

**商业化路径**
- **硬件定价**：1,800元（含腹带和指夹配件）+ 咬嘴耗材（20元/个，建议每月更换）
- **服务订阅**：129元/月（云端处方管理+医生随访+月度报告）
- **渠道**：呼吸科/全科医院处方推荐、COPD患者社群、电商
- **保险合作**：慢病管理险种配套设备（COPD是主要慢病险标的）
- **B端**：养老院批量配置（社区肺康复站）

**技术风险评估**

| 风险 | 等级 | 应对措施 |
|------|------|----------|
| 老年人呼吸方式不一致，气流测量受干扰 | 高 | 指导患者正确使用咬嘴，前5次训练有"姿势校正"引导 |
| SpO2低值老年人（慢性缺氧）误触停止 | 中 | 医生设置个性化SpO2阈值（部分COPD患者基线本就<90%） |
| 阻力阀机械可靠性（长期使用） | 中 | 舵机+减速箱设计，寿命>10,000次动作，质保2年 |
| 3D渲染对老年人手机性能要求 | 中 | 提供低配模式（2D圆形波形图），兼容5年以上旧款手机 |
| 用户坚持使用率低（慢性病依从性普遍差） | 中 | "呼吸作品"收集机制（集满30天解锁特殊造型），游戏化激励 |
