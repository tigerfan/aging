## 产品体系概览

```
认知健康（P01–P06）
  └─ 脑波/睡眠/情绪/行为/叙事/嗅觉 → 认知干预与评估

运动康复（P07–P12）
  └─ 步态/肌力/平衡/手部/步行/呼吸 → 功能恢复与预防

慢病管理（P13–P18）
  └─ 血糖/血压/心律/用药/肾病/骨密度 → 慢病监测与干预

心理社交（P19–P23）
  └─ VR疗愈/生命叙事/家庭连接/情绪监测/社群匹配 → 心理健康与社会连接

居家安全（P24–P27）
  └─ 跌倒监测/营养管理/睡眠监测/光疗 → 居家环境安全

医养连接（P28–P30）
  └─ 远程问诊/健康档案/照护协同 → 医疗资源可及性提升
```
