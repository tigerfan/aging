# P14 血压贴片——连续无袖带血压监测贴片

---

## A. 需求分析文档

### 1. 产品概述与定位

**血压贴片**是一款贴附于手腕内侧或锁骨下方的柔性电子贴片，通过PTT（脉搏传导时间）技术结合机器学习算法，实现连续无袖带血压监测。每小时自动采集一次血压，夜间血压谷值、晨峰血压等关键指标全自动记录，为老年高血压患者提供完整的24小时血压动态图谱，帮助医生精准评估心血管风险并优化用药方案。

**蓝海点**：全球高血压患者13亿，现有24小时动态血压监测（ABPM）须在诊所安装，极不便；Apple Watch/三星手环的血压功能精度未达医疗级；专用医疗级连续血压贴片市场几乎空白。

### 2. 目标用户画像

| 维度 | 描述 |
|------|------|
| 核心用户 | 60–80岁，高血压患者，需优化降压药用药方案 |
| 次要用户 | 白大衣高血压、隐匿性高血压患者（诊室外血压异常） |
| 使用场景 | 连续佩戴3–7天，全天候监测（含睡眠、运动） |
| 医疗背景 | 心内科/全科医生随访，替代传统ABPM |
| 关注指标 | 晨峰血压、夜间血压下降率（杓型/非杓型）、血压变异性 |

### 3. 市场痛点与机会

- 中国高血压患者2.45亿，血压控制达标率仅16.8%
- 传统ABPM：诊所操作、24小时戴袖带、影响睡眠、仅获得一次数据
- 家用血压计：每日2次，无法捕捉晨峰血压和夜间低谷
- 研究证实：夜间血压和晨峰血压是心血管事件的独立预测因子
- 连续血压监测数据可精准指导降压药品种选择和服药时间

### 4. 功能性需求（FR）

**FR-01 连续血压监测**
- PTT法：双光电探测器测量同一动脉脉搏波的传导时间差
- 测量频率：每1小时自动测量，可临时触发即时测量
- 输出：收缩压（SBP）、舒张压（DBP）估算值，±5mmHg精度（校准后）
- 夜间静默模式：不发出任何声音/振动，不影响睡眠

**FR-02 个性化校准**
- 首次使用：同步标准袖带血压计测量5次，建立个人PTT-血压对应模型
- 每周自动提醒用袖带血压计校准一次（防止模型漂移）
- 校准质量评分：实时显示模型置信度

**FR-03 血压动态图谱分析**
- 24小时血压曲线（时间-血压折线图）
- 自动分析：晨峰血压（起床后2小时内峰值）、夜间血压下降率、杓型/非杓型判断
- 血压变异性指标：标准差、变异系数、ARV（连续读数差均值）
- 高血压危象预警：SBP>180mmHg或DBP>120mmHg持续15分钟 → 告警

**FR-04 用药效果评估**
- 用药记录（时间、药物、剂量）与血压曲线叠加显示
- 自动计算：用药后血压下降幅度、降压谷值时间（谷峰比）
- 定期报告：降压药疗效评分，供医生调整处方参考

**FR-05 医生随访数据**
- 导出ABPM标准报告格式（PDF）
- 医生端Web查看患者血压趋势
- 异常血压事件推送给主治医生

### 5. 非功能性需求（NFR）

| 维度 | 指标 |
|------|------|
| 血压精度 | SBP误差≤5mmHg，DBP误差≤5mmHg（校准后，符合IEEE 1708标准） |
| 贴片续航 | ≥7天（无线充电更换或可更换电池设计） |
| 厚度 | ≤2mm（柔性贴片，不影响日常动作） |
| 粘附性 | 医用级压敏胶，可持续7天粘附，撕除无痛 |
| 防水 | IPX7（可淋浴） |
| 皮肤安全 | ISO 10993认证生物相容性，过敏测试合格 |

### 6. 约束条件与合规要求

- 作为**II类医疗器械**（无创血压测量设备）申报，须通过IEEE 1708临床验证
- 严重心律不齐患者（房颤）：PTT法精度大幅下降，须特别声明
- 产品定位：辅助监测工具，不替代临床血压测量用于诊断
- 皮肤粘附：持续7天须无皮肤刺激，提供过敏测试套件

## B. 详细设计方案

### 1. 系统架构

```

     ├── ECG单导联电极（可选，ECG-PTT更准确）
     ├── 加速度计（运动伪迹检测）
     ├── 皮肤温度传感器
     └── 超低功耗MCU + BLE

          ▼
[手机APP]
     ├── 血压模型推理（本地）
     ├── 动态血压图谱
     ├── 用药记录管理
     └── 医生报告
          ▼
[云端]
     ├── 个性化模型训练
```

### 2. 硬件设计方案

**2.1 传感器设计**
- 双PPG：两个绿光LED（520nm）+ 光电二极管，间距20mm，测量脉搏波传导时间
- 可选单导联ECG：两个Ag/AgCl干电极，配合ECG芯片（ADS1292）
- 加速度计：LIS3DH（MEMS，超低功耗），用于检测运动、排除运动伪迹
- 温度传感器：NTC热敏电阻，±0.1°C精度

**2.2 主控与通信**
- MCU：Silicon Labs EFM32（超低功耗Cortex-M0+，睡眠电流<1μA）
- BLE：Nordic nRF52810（低功耗BLE 5.0），每小时唤醒发送一次数据
- 电池：印刷薄膜锂电池（容量30mAh），蓝牙充电（NFC无线充电）

**2.3 柔性贴片结构**
- 基材：聚酰亚胺（PI）柔性PCB，弯曲半径>5mm
- 封装：医用级硅胶覆膜，防水防汗
- 粘附层：3M 2477P医用压敏胶（双面），7天持续粘附力
- 尺寸：40mm×25mm×1.8mm，重量<5g

### 3. 核心算法设计

**3.1 PTT血压估算**
```python
def estimate_bp_from_ptt(ptt_ms, personal_model):
    # PTT与血压呈负相关（血压升高，脉搏波传导加快）
    # 个人模型：y = a * exp(-b * PTT) + c
    sbp = personal_model.predict_sbp(ptt_ms)
    dbp = personal_model.predict_dbp(ptt_ms)
    return sbp, dbp

def extract_ptt(ppg1, ppg2, fs=200):
    # 检测两路PPG的波峰时间差
    peak1 = find_peaks(ppg1, distance=fs//2)
    peak2 = find_peaks(ppg2, distance=fs//2)
    ptt = mean(peak2 - peak1) / fs * 1000  # ms
    return ptt
```

**3.2 运动伪迹自动剔除**
- 加速度计检测：若加速度RMS > 0.3g，标记为"运动期"，不纳入血压统计
- 呼吸伪迹：PPG信号中0.15–0.4Hz成分用于检测呼吸调制，排除呼吸不规律导致的PTT误差

**3.3 晨峰血压与杓型分析**
```
晨峰血压 = 起床后2小时内SBP均值 - 睡眠期SBP最低值
夜间血压下降率 = (日间SBP均值 - 夜间SBP均值) / 日间SBP均值 × 100%
杓型判断：下降率>10% → 杓型（正常），<10% → 非杓型（高风险）
```

### 4. 用户交互设计

**使用流程**
```
撕下贴片保护膜 → 贴于手腕内侧桡动脉处
→ APP蓝牙自动识别 → 引导用袖带血压计测量5次（校准）
→ "校准完成，开始连续监测"
→ 每1小时自动采集，APP实时更新血压曲线
→ 每晨推送"昨夜血压报告"：夜间均值、晨峰值、杓型状态
→ 7天后自动生成ABPM报告，一键分享给医生
```
