# P04 认知镜——日常行为认知评估镜

---

## A. 需求分析文档

### 1. 产品概述与定位

**认知镜**是一款外形如普通浴室镜的智能设备，通过毫米波雷达和镜面内嵌屏幕，在用户每日洗漱/梳妆的自然场景中，无感完成行为认知评估（洗手步骤完整性、反应时间、双手协调性等），持续追踪认知功能变化趋势，早期识别认知衰退信号，同时向家属推送评估摘要。

**核心创新**：零主动操作、无摄像头（保护隐私）、嵌入日常生活场景，解决老年人认知评估依从性极差的根本问题。

### 2. 目标用户画像

| 维度 | 描述 |
|------|------|
| 核心用户 | 65–80岁，独居或日间独处的老年人 |
| 次要用户 | 有痴呆家族史、主动进行认知监测的55–65岁人群 |
| 场景 | 每日洗漱/梳妆，5–15分钟，自然习惯性行为 |
| 关注方 | 成年子女、社区养老服务人员、家庭医生 |
| 技术门槛 | 接近零，用户完全无感知，无需配合 |

### 3. 市场痛点与机会

- 认知评估量表（MMSE、MoCA）需专业人员实施，社区无法高频普查
- 现有可穿戴设备（智能手表）专注心率/步数，认知评估能力空白
- 患者自我感知迟滞：通常认知损害已达中度时才被发现，错过早期干预窗口
- 浴室镜是家庭渗透率最高的日用品之一，改造为智能设备接受度高
- 毫米波雷达不采集图像，隐私风险极低，监管阻力小

### 4. 功能性需求（FR）

**FR-01 无感行为捕捉**
- 77GHz毫米波雷达阵列捕捉用户洗漱行为：
  - 手部运动轨迹（洗手步骤顺序与完整性）
  - 面部接近与离开镜面的距离变化
  - 站立稳定性（躯干微动检测）
  - 双手协调动作（如刷牙、梳头）
- 检测范围：镜前0.3–1.5m，横向覆盖宽度1.2m

**FR-02 认知功能评估维度**
- 执行功能：洗手/刷牙的步骤完整性与顺序正确性
- 反应时间：对镜面轻柔提示（如光线变化）的响应延迟
- 空间感知：手部动作精准度（如接近水龙头的定位误差）
- 运动协调：双手动作一致性评分

**FR-03 趋势分析与预警**
- 每日评分存入数据库，按周/月生成趋势图
- 连续3周某维度得分下降超15%，推送"建议关注"预警
- 年度认知变化对比报告（可附带MoCA量表同期数据）

**FR-04 镜面交互（辅助功能）**
- 镜面内嵌透明屏，正常使用时完全透明（看不出是屏幕）
- 当检测到用户在镜前时，右侧角落显示：时间、今日天气、当日用药提醒
- 简单手势控制（大幅挥手）：切换信息显示/隐藏

**FR-05 家属端报告**
- 每日自动生成评估摘要推送家属APP
- 异常预警立即推送（附带建议：关注/就医咨询）
- 月度报告PDF可发送给家庭医生

**FR-06 生活安全监测（附加）**
- 跌倒检测：雷达检测到人体急速下落信号，触发告警
- 长时间静止：在镜前静止超过5分钟，推送确认提醒
- 异常时间：深夜（1–4点）出现洗漱行为，推送家属（可能为认知障碍游荡行为）

### 5. 非功能性需求（NFR）

| 维度 | 指标 |
|------|------|
| 外观 | 与普通浴室镜无法区分，无外露线缆 |
| 评估准确性 | 行为步骤识别准确率≥88% |
| 隐私保护 | 无摄像头，雷达数据不包含图像，仅点云数据 |
| 防水防雾 | 镜面加热防雾，整机IP44防水（浴室环境） |
| 安装 | 专业安装（替换现有浴室镜），安装时间<1小时 |
| 可靠性 | 浴室高温高湿环境下MTBF>20,000小时 |
| 延迟 | 行为识别到显示反馈延迟<500ms |

### 6. 约束条件与合规要求

- 毫米波雷达需取得MIIT型号核准证（型号核准）
- 作为**健康管理设备**申报，不标注诊断功能
- 镜面加热功率≤40W，安全低压（24V DC）
- 符合家用电器安全标准GB 4706，防水等级IP44
- 数据存储：评估数据匿名化处理后可用于模型改进，须用户知情同意

## B. 详细设计方案

### 1. 系统架构

```
[认知镜设备]
     ├── 77GHz毫米波雷达阵列（3发4收）
     ├── 透明OLED/LCD显示层（嵌入镜面后方）
     ├── 镜面加热膜（防雾）
     ├── 扬声器（语音提醒，低功率）
     └── 边缘计算模块（行为识别推理）
          │ WiFi 6 / 以太网
          ▼
[云端服务]
     ├── 认知评估数据库
     ├── 趋势分析引擎
     ├── 报告生成服务
     └── 预警推送服务
          ▼
[家属APP / 家庭医生门户]
```

### 2. 硬件设计方案

**2.1 毫米波雷达系统**
- 芯片：TI AWR1642（77GHz FMCW雷达，2发4收）
- 最大距离分辨率：4cm，角度分辨率：15°
- 点云数据：每帧50–200个点，帧率25fps
- 安装位置：镜面顶部中央内嵌，水平安装

**2.2 显示系统**
- 方案：半透明LCD+镜面贴膜（成本优）或透明OLED（高端版）
- 显示区域：镜面右侧20%区域，可动态开关
- 亮度自适应：根据浴室环境光强度调节（50–300nit）
- 防雾加热：PTC加热膜，温控55–60°C，防水接插件连接

**2.3 边缘计算模块**
- 单板计算机：瑞芯微RK3568（四核A55，0.8TOPS NPU）
- 内存：2GB LPDDR4 + 32GB eMMC
- 运行自定义Linux系统，Docker容器化部署推理服务

**2.4 结构与安装**
- 尺寸：60cm×80cm（标准浴室镜尺寸），厚度3.5cm
- 安装：挂墙安装，隐藏式电源走线（预置电源接口）
- 材料：镜框不锈钢拉丝，防腐蚀处理（IP44）

### 3. 软件设计方案

**3.1 雷达信号处理流水线**
```
原始ADC数据
    → 距离FFT（Range FFT）
    → 速度FFT（Doppler FFT）
    → CFAR目标检测
    → 点云聚类（DBSCAN）
    → 人体骨架估计（点云→关节点映射）
    → 行为识别模型推理
    → 评估指标计算
```

**3.2 行为识别模型**
- 输入：连续2秒点云序列（50帧×每帧点云，转化为体素网格）
- 架构：Point Transformer（轻量化，适配NPU，参数量<2M）
- 输出：当前行为类别（洗手/刷牙/梳头/洗脸/静止/离开）+ 步骤完整性评分
- 数据集：采集200名老年人标准行为视频，同步雷达数据，标注行为步骤

**3.3 认知评估指标计算**

| 指标 | 计算方法 |
|------|----------|
| 洗手步骤完整性 | HMM序列匹配，与标准7步洗手法比对，计算步骤覆盖率 |
| 反应时间 | 从用户进入检测区到开始执行洗漱动作的延迟（基线对比） |
| 运动流畅度 | 手部运动轨迹的平滑度（加加速度/Jerk指标） |
| 双手协调性 | 双手运动轨迹时间互相关系数 |
| 综合认知评分 | 加权平均（执行功能40% + 反应时间30% + 协调性30%） |

**3.4 镜面UI系统**
- 轻量Web引擎（Chromium Kiosk模式）渲染镜面显示
- 内容：数字时钟（大字体）、天气（图标+温度）、用药提醒（到时高亮）
- 人体检测触发：检测到用户后0.5秒显示，用户离开后5秒渐隐

### 4. 核心算法设计

**4.1 洗手步骤识别（HMM方法）**
- 定义7个洗手状态：S1手背搓揉→S2手心搓揉→S3指缝清洁→S4指背搓揉→S5拇指旋转→S6指尖搓揉→S7手腕搓揉
- 观测序列：每秒雷达提取的手部运动模式特征向量
- 训练：使用Baum-Welch算法在50名老年人数据上训练HMM参数
- 推断：Viterbi算法解码最优状态路径，计算步骤覆盖率

**4.2 认知衰退预警算法**
```
每日：计算综合认知评分 S(t)
每周：
  trend = linear_regression(S(t-20), ..., S(t))  # 20天趋势
  if trend.slope < -0.5 per week:  # 每周下降>0.5分
    if consecutive_weeks >= 3:
      send_alert(level="建议关注", message="近3周认知评分持续下降")
  
  if S(t) < S(t-90) * 0.85:  # 较3个月前下降>15%
    send_alert(level="建议就医", message="认知功能较3个月前显著下降")
```

**4.3 个体基线自适应**
- 新用户前30天为"基线建立期"，采集评估数据建立个体基准
- 后续评估基于个体基线进行相对比较（不与群体平均值比较）
- 季节/健康状态因素：节假日、生病期间数据标记为"特殊状态"，不纳入趋势计算

### 5. 用户交互设计

**5.1 老人端（极简，几乎无感知）**
- 用户正常洗漱，设备在后台运行
- 仅在以下情况给予轻柔提示：
  - 用药提醒：镜面右上角闪烁药丸图标+轻柔音效
  - 步骤遗漏提示（可选开关）："还没有清洁手腕哦"（温和语气）
  - 跌倒告警：急促音效+家属自动呼叫

**5.2 家属APP体验**
- 主界面：父母今日认知评分大卡片 + 本周趋势小图
- 通知：每日早9点推送昨日摘要，仅预警级别才实时推送
- 月报：PDF附件，可一键转发给家庭医生

**5.3 安装引导（上门服务）**
- 专业安装师上门，替换现有浴室镜（45–60分钟）
- 安装后运行10分钟校准程序（用户站在镜前自然活动）
- 家属远程配置（APP端完成）

### 6. 商业化路径与技术风险

**商业化路径**
- **产品形态**：标准版（3,600元，含安装）/ 高端版透明OLED（5,800元）
- **服务费**：180元/年（云端分析+报告+预警服务）
- **渠道**：
  - B端：智慧社区养老项目采购（政府补贴），批量安装
  - C端：高端装修/适老化改造配套
  - 保险公司：作为健康管理权益提供给老年保单持有人
- **数据价值**：脱敏认知行为数据可授权给药企/医院用于临床研究（额外收入）

**技术风险评估**

| 风险 | 等级 | 应对措施 |
|------|------|----------|
| 毫米波雷达在复杂浴室环境多径干扰 | 高 | 专用浴室场景数据集训练，多帧平均去噪 |
| 认知评估指标与临床量表相关性弱 | 高 | 联合医院进行相关性研究，至少收集500对数据 |
| 浴室高温高湿导致硬件故障 | 中 | IP44认证，关键器件额外封胶处理，延长质保至3年 |
| 安装复杂度影响推广速度 | 中 | 培训第三方安装团队，覆盖50个城市 |
| 隐私敏感（浴室场景） | 中 | 强调无摄像头，仅雷达点云，发布独立隐私白皮书 |
