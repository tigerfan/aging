# P20 数字遗嘱仪——生命叙事与数字遗产管理平台

---

## A. 需求分析文档

### 1. 产品概述与定位

**数字遗嘱仪**是一款帮助老年人整理人生故事、记录重要嘱托、管理数字遗产的软硬件系统。通过专为老年人设计的桌面录制终端，引导老人以视频/音频/文字形式记录人生回忆、家族故事、给亲人的话和财产安排，AI自动整理归档，形成结构化的"人生档案库"，在老人希望的时间以希望的方式传递给指定家庭成员。

**蓝海点**：数字遗嘱和生命叙事（Life Review）在心理学上被证明可显著改善老年人生命意义感和死亡焦虑；中国老龄人口对传统遗嘱认知不足，数字遗产管理意识基本空白；具有情感价值的生命叙事产品市场空白。

### 2. 目标用户画像

| 维度 | 描述 |
|------|------|
| 核心用户 | 65–85岁，思维清晰，有心理整合需求，愿意留下人生记录的老年人 |
| 次要用户 | 重病/临终患者（安宁疗护场景）的生命叙事需求 |
| 家属 | 希望留存父母/祖父母的声音和故事，作为家族记忆传承 |
| 情感动机 | 留下印记、被记住、完成心愿、减少遗憾 |
| 文化背景 | 中国老年人普遍存在生命叙事冲动，但缺乏合适工具 |

### 3. 市场痛点与机会

- 数字时代的老年人留有大量分散的照片、视频、社交内容，但几乎无人整理
- 传统遗嘱仅处理财产，无法承载情感和故事
- 国外Life Review产品（StoryWorth、Eternal等）：无中文本地化版本，且缺乏硬件录制终端
- 死亡教育在中国逐渐兴起（安宁疗护政策推进），生命叙事工具有政策支撑
- "代际数字鸿沟"使得老年人的大量经历和智慧正在消失，家庭情感价值极高

### 4. 功能性需求（FR）

**FR-01 录制终端（硬件）**
- 10英寸平板+支架，桌面放置，类似"情感相框"外观
- 前置摄像头（4K，自动美化，柔光补光）+ 麦克风阵列（降噪）
- 一键录制模式：按下专用录制按键即开始，无需复杂操作
- 屏幕显示：引导词提示（大字体）+当前录制状态

**FR-02 AI引导叙事**
- 内置200+生命叙事引导问题（分类：童年回忆、人生转折、爱与家庭、人生智慧、给亲人的话等）
- LLM智能追问：根据老人回答，自动生成个性化追问（"您说小时候家在山里，那时候冬天怎么取暖？"）
- 语音识别+实时字幕：老人口述，屏幕同步显示文字，便于确认
- 每次录制建议15–20分钟，轻松节奏

**FR-03 内容整理与归档**
- 语音自动转文字（ASR，准确率>95%，方言支持）
- AI自动提炼关键故事片段、摘要和标签
- 按主题自动归类（童年/事业/家庭/智慧/嘱托）
- 时间轴视图：人生故事按年代排列的可视化时间轴

**FR-04 数字遗产管理**
- 账号清单：老人录入各类账户（银行、微信、支付宝、社交媒体等）和密码，加密存储
- 数字内容：照片、视频、重要文件的整理和分类
- 遗嘱声明：视频/文字遗嘱录制（不具法律效力，建议同时办理公证，系统提供指引）
- 权限管理：指定受益人，设定"到时开启"条件（如老人去世后30天）

**FR-05 传递与共享**
- 立即分享：选定内容立即分享给指定家属（视频片段、故事摘要）
- 定时胶囊：设定特定日期开启（孙子18岁、结婚纪念日等），自动推送
- 纸质成册：年度将所有故事整理成精装相册（印刷服务，额外付费）
- 家族树功能：多代家庭成员共同贡献故事，形成家族档案

### 5. 非功能性需求（NFR）

| 维度 | 指标 |
|------|------|
| 录制视频质量 | 4K 30fps，内置补光使面部清晰，背景虚化 |
| 语音识别准确率 | 标准普通话>97%，方言>88% |
| 数据安全 | 端对端加密，数据托管于受托机构（类似律师事务所数据保管） |
| 存储容量 | 云端每用户100GB起（约200小时视频） |
| 遗嘱声明有效性 | 提供与公证处合作渠道，电子遗嘱经公证后具法律效力 |
| 账号密码安全 | AES-256加密，受益人需验证身份（人脸+短信）才能解锁 |

### 6. 约束条件与合规要求

- 账号密码数据须本地加密存储，云端仅存储加密密钥的哈希，任何人（含厂商）不可解密
- 遗嘱视频须明确告知：不具备法律效力，仅为情感记录，财产遗嘱须公证
- 受益人身份核验须多重认证（人脸识别+手机+身份证）
- "到时开启"机制须有第三方见证（公证处/律师事务所合作），防止纠纷
- 用户去世后，数据按照预设指令执行，厂商不得单方面访问数据

## B. 详细设计方案

### 1. 系统架构

```
[数字遗嘱仪终端（桌面平板）]
     ├── 4K前置摄像头 + 补光灯
     ├── 4麦克风降噪阵列
     ├── 10英寸触控屏
     ├── 专用录制/确认大按键
     └── SBC主控 + WiFi
          │ HTTPS（E2E加密）
          ▼
[安全云端（数据托管）]
     ├── 视频存储（E2E加密）
     ├── ASR转录服务
     ├── LLM叙事引导引擎
     ├── AI内容整理（摘要/分类/标签）
     └── 数字遗产保险箱（加密密钥管理）
          ▼
[家属APP（受益人端）]
     ├── 授权内容查看
     ├── 定时胶囊管理
     └── 家族树贡献
```

### 2. 硬件设计方案

**2.1 录制终端**
- 主板：Raspberry Pi 5（4GB RAM，足够4K视频处理）
- 显示：10.1英寸IPS触控屏（1920×1200），字体≥24pt
- 摄像头：索尼IMX678（4K@30fps），F1.8大光圈，自动美颜（肤色提亮、微磨皮）
- 补光：环形LED补光灯（5000K自然白，亮度自动调节），减少面部阴影
- 外观：实木边框+亚光白面板，类似高档相框，可挂墙或桌面支架

**2.2 录制按键设计**
- 大圆形录制键（直径40mm，红色触感），单次按下开始录制，再次按下结束
- 确认键（绿色，"这次录得好，保存"）
- 重录键（黄色，"再来一次"）
- 三键操作完成整个录制流程，无需触屏

### 3. 软件设计方案

**3.1 引导叙事系统（LLM驱动）**
```
系统提示词（System Prompt）：
你是一位温暖、耐心的生命叙事引导者，正在帮助一位中国老人记录人生故事。
规则：
1. 使用简单、温暖的语言，避免专业词汇
2. 每次只问一个问题，等待完整回答
3. 对回答中的细节给予真诚的追问（"那您当时心里是什么感受？"）
4. 对所有回答保持尊重和好奇，不评判
5. 每次录制主题聚焦，不发散

引导问题库（按主题分类）：
童年记忆（50题）："您小时候最喜欢的游戏是什么？"
人生转折（40题）："哪个决定改变了您的一生？"
爱与家庭（60题）："您和老伴第一次见面是什么情形？"
智慧传承（30题）："您最想告诉年轻人的一句话是什么？"
给亲人的话（20题）："对您的孩子，有什么话一直想说但没说的？"
```

**3.2 AI内容整理管道**
```
原始视频 → ASR转录（全文字幕）
    ↓
关键信息提取（人名/地名/时间/事件 NER识别）
    ↓
故事摘要生成（100字精华提炼）
    ↓
情感标签分析（快乐/感恩/遗憾/骄傲/思念）
    ↓
时间轴定位（推断事件发生年代）
    ↓
人生时间轴自动更新
    ↓
家庭摘要推送（"奶奶今天讲了一段关于1970年代的故事"）
```

**3.3 数字遗产保险箱**
- 密码存储：AES-256-GCM加密，密钥由用户设定的"记忆密语"派生（KDF），不可恢复（厂商无法访问）
- 解锁条件：支持"时间解锁"（设定日期）和"事件解锁"（由两位受益人共同申请+身份核验）
- 密钥托管：支持与公证处合作，密钥分片（Shamir Secret Sharing），由律师事务所保管一片

### 4. 核心算法设计

**4.1 人生时间轴自动构建**
```python
def build_life_timeline(all_transcripts):
    events = []
    for transcript in all_transcripts:
        # NER提取时间信息（"1978年"、"改革开放那年"、"我女儿出生时"）
        time_entities = ner_extract_time(transcript)
        # 规则+LLM推断实际年份
        year = infer_year(time_entities, user_birth_year)
        # 事件摘要
        event_summary = llm_summarize(transcript, max_tokens=50)
        events.append({'year': year, 'summary': event_summary, 'clip_id': transcript.id})
    
    return sorted(events, key=lambda x: x['year'])
```

**4.2 情感分析（本土化）**
- 模型：基于中文BERT微调的情感分类模型（6类：快乐/感恩/遗憾/骄傲/思念/平静）
- 用途：情感标签帮助家属了解老人的情感状态，也帮助对话系统调整引导方向

### 5. 用户交互设计

**录制流程**
```
打开设备（按电源键，大字欢迎界面）
→ 今日引导问题展示（大字体，屏幕中央）
   "今天想请您聊聊：您在哪里长大的？那个地方最美的地方是哪里？"
→ 按红色录制键 → 开始说
→ AI自动追问展示在屏幕上（"您提到了河边，那时候河里有鱼吗？"）
→ 按绿色保存键 → "已保存，这段故事真精彩！"
→ 设备自动上传、整理，推送摘要给家属
```

**技术风险评估**

| 风险 | 等级 | 应对措施 |
|------|------|----------|
| 老人对"遗嘱"话题心理抵触 | 高 | 产品名称强调"生命记录/家族故事"，弱化遗嘱标签，以"留故事"切入 |

| 方言识别不准确 | 中 | 显示实时字幕供老人确认，支持手动纠正，不准确片段人工标注模式 |
