# P19 虚拟花园——VR冥想放松疗愈系统

---

## A. 需求分析文档

### 1. 产品概述与定位

**虚拟花园**是一款专为老年人设计的轻量化VR疗愈系统，通过4K沉浸式虚拟自然环境（四季花园、海边日落、森林清晨等），结合正念冥想引导、呼吸训练、音乐疗法，帮助老年人缓解慢性疼痛、焦虑和抑郁情绪。专为老年人优化的VR头显（重量轻、视野清晰、防晕眩设计），以及超简操作界面，使其成为日常心理健康护理的有效工具。

**蓝海点**：老年人心理健康被严重忽视；VR疗愈已有大量临床证据证明对疼痛缓解和焦虑改善有效；现有VR产品全部面向年轻用户，老年版设计几乎为零。

### 2. 目标用户画像

| 维度 | 描述 |
|------|------|
| 核心用户 | 65–80岁，有慢性疼痛、焦虑、抑郁倾向，或独居孤独感强的老年人 |
| 次要用户 | 养老院/安宁疗护病房患者，需要非药物疼痛管理 |
| 使用场景 | 每日1次，每次15–30分钟，坐姿使用，居家或养老院 |
| 情感需求 | 渴望与自然接触（行动不便老人）、内心宁静、有人陪伴的感觉 |
| 技术门槛 | 极低，一键开机即进入体验，无需任何操作学习 |

### 3. 市场痛点与机会

- 中国老年人抑郁症患病率20–30%，焦虑障碍15–20%，远超一般人群但就诊率极低
- 慢性疼痛（关节炎、腰背痛）困扰70%以上老年人，非药物干预需求迫切
- VR镇痛/放松疗法已通过多项RCT验证（Cedars-Sinai, AppliedVR等研究）
- 现有VR设备（Meta Quest等）对老年人不友好：重、操作复杂、字小
- 养老院非药物干预工具极度匮乏，护理人员希望有辅助工具

### 4. 功能性需求（FR）

**FR-01 老年人专用VR头显**
- 重量<300g（比Meta Quest 3轻约50%，减少颈部负担）
- 视野（FOV）：100°，避免老年人FOV焦虑感
- 防晕眩：固定场景模式（无移动视角），瞳距物理可调（56–72mm）
- 焦距调节：-6.0到+2.0屈光度调节，适应老花/近视无需额外眼镜
- 一键开机：按下侧面大按钮即进入上次体验场景

**FR-02 疗愈内容库**
- 自然场景：30个4K/360°沉浸场景（春季樱花园、夏日荷塘、秋山红叶、冬雪村庄、海边日出等）
- 引导冥想：12套正念冥想课程（10–20分钟，中文温柔女声引导）
- 音乐疗法：binaural beats（双耳节拍）搭配自然声音（鸟鸣/流水/风声）
- 呼吸训练：VR场景中融入可视化呼吸引导（荷花随呼吸开合）
- 内容每月更新（节气主题：清明踏青、中秋赏月等）

**FR-03 生理反馈监测**
- 头显内置：心率监测（PPG，太阳穴接触式）+ 皮肤电导（GSR，额头接触式）
- 放松效果实时反馈：HRV和GSR共同评估放松状态，场景背景色随放松程度变化（焦虑→蓝天明亮，越放松→暖色夕阳）
- 每次体验后：心率/HRV变化报告，放松效果评分（0–10）

**FR-04 个性化推荐**
- AI推荐系统：根据老人当日情绪状态（APP简单问卷或语音评估）推荐最合适场景
- 情绪评估：7点情绪量表（图形化，简单操作），记录情绪变化趋势
- 家属查看：情绪趋势图（月度），及时发现情绪恶化

**FR-05 安宁疗护特别功能**
- 个人化场景定制：上传老人年轻时钟爱的地方/家乡照片，AI生成虚拟重访场景
- 亲人声音集成：家人录音（语音信息），在VR场景中以"鸟鸣/风铃"形式出现（隐性陪伴感）

### 5. 非功能性需求（NFR）

| 维度 | 指标 |
|------|------|
| 晕动症发生率 | <5%（固定场景设计，无移动视角） |
| 分辨率 | 每眼≥2560×2560（避免纱门效应） |
| 头显重量 | <300g |
| 内容加载时间 | <5秒进入场景 |
| 续航 | ≥3小时（单次使用15–30分钟，3天不充电） |
| 卫生 | 面罩可拆卸清洗，配备3块替换面罩 |

### 6. 约束条件与合规要求

- 严重青光眼患者：VR头显产生的闭合空间和光刺激可能增加眼压，须标注
- 使用时长限制：建议单次不超过30分钟（避免眼睛疲劳）
- 疗愈场景定位：明确为"身心放松"工具，非医疗器械，不作治疗主张
- 个人化场景（老家照片）：数据本地处理，不上传云端

## B. 详细设计方案

### 1. 系统架构

```
[虚拟花园VR头显]
     ├── 双2.5K AMOLED Micro显示屏
     ├── 6DoF位置追踪（仅旋转，不移动）
     ├── PPG心率传感器（太阳穴）
     ├── GSR皮肤电导（额头）
     ├── 骨传导扬声器（无需耳机，不隔绝外部环境）
     ├── SBC计算板（骁龙XR2，运行VR内容）
     └── WiFi + BLE（内容同步）
          │ WiFi
          ▼
[配套APP（家属/护理端）]
     ├── 情绪趋势监控
     ├── 内容库管理
     └── 个人化场景上传
          ▼
[云端内容分发]
     ├── 月度新场景推送
     └── 个人化AI场景生成
```

### 2. 硬件设计方案

**2.1 头显核心**
- SoC：高通骁龙XR2（专为VR优化，支持4K/90fps）
- 显示：双2.5K AMOLED（2560×2560/眼），刷新率90Hz，90fps防闪烁
- 镜片：菲涅尔透镜（重量轻），电调焦镜片（电动驱动，APP控制焦距）
- 追踪：6DoF旋转追踪（陀螺仪+加速度计），仅追踪头部转向，不追踪移动（防晕眩）

**2.2 生理传感**
- PPG：绿光LED+光电二极管，贴合太阳穴，采样率100Hz
- GSR：两个Ag触点（额头），测量皮肤电导率（μS），放松时GSR下降

**2.3 音频系统**
- 骨传导振动器（贴合颅骨两侧），传递低频振动（心跳感、低音），保留自然环境感知
- 扬声器：2个5mm喇叭（近耳式，非密闭），自然声+音乐疗法

**2.4 结构与人体工学**
- 重量：主体260g（对比Quest 3的515g），重心后移（颈部负担降低）
- 面罩：可换洗硅胶面罩，4磁点快拆，附3块替换
- 前额支撑带（辅助分担重量），松紧一键调节

### 3. 核心算法设计

**3.1 放松状态评估**
```python
def compute_relaxation_score(hr_series, hrv_series, gsr_series):
    # HRV上升 + HR下降 + GSR下降 → 放松
    hr_change = (baseline_hr - current_hr) / baseline_hr  # 正值=心率降低
    hrv_change = (current_hrv - baseline_hrv) / baseline_hrv  # 正值=HRV增加
    gsr_change = (baseline_gsr - current_gsr) / baseline_gsr  # 正值=GSR降低
    
    score = (hr_change * 0.3 + hrv_change * 0.5 + gsr_change * 0.2) * 100
    return max(0, min(100, score))  # 0–100分
```

**3.2 场景氛围自适应调整**
- 放松分数>60：场景色调暖化（黄昏金色光），声音音量小，节奏慢
- 放松分数<30（仍焦虑）：切换为更舒缓场景（流水声+静态花园），并加入深呼吸引导

**3.3 个人化AI场景生成**
- 输入：老人上传的家乡/年轻时钟爱地方的照片（5–10张）
- 处理：Stable Diffusion ControlNet生成360°全景图（照片风格迁移到四季场景）
- 输出：专属"记忆花园"场景（本地生成，不上传）

### 4. 用户交互设计

**使用流程（一键体验）**
```
佩戴头显（面罩贴合，松紧调节）
→ 按侧面大按钮 → 系统自动打开上次场景
→ 骨传导振动轻微震动（提示已连接）
→ "欢迎来到您的花园，今天为您准备了春日樱花..."
→ 完全沉浸，跟随引导呼吸和冥想
→ 30分钟后语音轻声提示 → 按钮结束
→ APP推送"今日放松评分85分，心率降低8次/分钟，很好！"
```

### 5. 商业化路径与技术风险

**商业化路径**
- **头显**：2,200元；**养老院版**（耐用款+管理系统）：3,600元/台
- **内容订阅**：99元/年（月度新场景+个人化场景生成服务）
- **B端**：养老院/安宁疗护机构批量采购（非药物干预工具）
- **保险**：心理健康险配套，作为预防性健康管理工具

**技术风险评估**

| 风险 | 等级 | 应对措施 |
|------|------|----------|
| 晕动症（即使固定场景） | 中 | 首次使用5分钟适应模式，缓慢过渡，FOV限制（90°以内） |
| 老年人佩戴眼镜困难 | 高 | 电调焦镜片覆盖-6.0至+2.0屈光度，大多数老人无需摘眼镜 |
| 内容枯燥（长期使用兴趣下降） | 中 | 月度新场景推送，节气主题（24节气=24个新场景/年） |
| 额头PPG/GSR传感稳定性 | 中 | 面罩内置弹性电极垫，确保与皮肤接触，适当放宽精度要求 |
