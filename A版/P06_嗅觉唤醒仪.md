# P06 嗅觉唤醒仪——气味疗法认知激活设备

---

## A. 需求分析文档

### 1. 产品概述与定位

**嗅觉唤醒仪**是一款将精准气味释放与脑电神经反馈结合的认知激活设备。通过微流控技术精准控制气味种类与浓度，在用户佩戴EEG头带的同时，依据实时脑电状态自适应释放不同气味（迷迭香提升专注、薰衣草促进放松、柠檬改善情绪），并结合气味触发的记忆再激活效应，为轻度认知障碍老年人提供非药物嗅觉疗法干预。

**蓝海点**：嗅觉疗法（Olfactory Therapy）有循证支持，但尚无软硬件结合的智能化设备，市场几乎空白。嗅觉与记忆的神经解剖联系（嗅球直连海马体）为产品提供坚实科学基础。

### 2. 目标用户画像

| 维度 | 描述 |
|------|------|
| 核心用户 | 60–80岁，轻度认知损害（MCI）或嗅觉减退老年人 |
| 次要用户 | 有认知干预需求的健康老年人 |
| 使用场景 | 每日1次，30分钟，居家客厅/卧室，坐姿使用 |
| 照护背景 | 可独立使用，也适用于专业照护机构集体疗法 |
| 关注点 | 自然无药物、有科学依据、操作简便 |

### 3. 市场痛点与机会

- 嗅觉减退是阿尔茨海默症的早期预警信号（早于记忆症状3–5年），但几乎无专用干预设备
- 芳香疗法市场混乱，缺乏科学剂量控制和个体化方案
- 现有香薰设备无传感器反馈，无法根据用户状态动态调整
- 微流控技术成熟（医疗级）且成本降低，具备消费化条件
- 养老机构对"非药物认知疗法"设备采购需求旺盛

### 4. 功能性需求（FR）

**FR-01 脑电状态监测**
- 单通道EEG头带（额叶贴合，干电极），实时采集脑电信号
- 提取α波功率（放松指标）、θ波功率（记忆指标）、总体认知激活水平
- 每30秒更新一次认知状态评估

**FR-02 精准气味释放**
- 内置8种精油卡匣：迷迭香、薰衣草、柠檬、薄荷、茉莉、松木、玫瑰、橙花
- 微流控泵精确控制释放流量：0.1–5μL/min，精度±5%
- 气味浓度可调（低/中/高三档）
- 混合气味配方：最多同时释放2种精油，按比例混合

**FR-03 闭环反馈控制**
- 根据脑电状态动态切换气味配方：
  - 认知激活不足（α波过高）→ 释放迷迭香/薄荷（提神激活）
  - 焦虑激动（β波过高）→ 释放薰衣草/橙花（镇定放松）
  - 记忆训练阶段 → 释放柠檬/茉莉（记忆关联气味）
- 气味切换间隔≥5分钟（嗅觉适应期）

**FR-04 嗅觉功能评估**
- 内置嗅觉辨别测试：程序化释放气味，用户通过APP选择气味名称
- 每月进行一次嗅觉敏感度评估，生成嗅觉功能趋势图
- 异常嗅觉下降时推送就医建议（嗅觉减退可能是AD早期信号）

**FR-05 记忆关联训练（TMR气味版）**
- 白天学习时，APP指导用户同时吸入特定气味（记忆编码阶段）
- 训练时将该气味设为记忆关联气味
- 正式疗法阶段释放同一气味，触发记忆再激活效应
- 配合语音提示，引导老人尝试回忆相关内容

**FR-06 疗程管理**
- 标准疗程：4周，每日1次，每次30分钟
- 疗程完成后生成认知改善评估报告（基线vs疗程后）
- 卡匣更换提醒（每卡匣约60次使用）

### 5. 非功能性需求（NFR）

| 维度 | 指标 |
|------|------|
| 气味释放精度 | 浓度控制误差<10%，重复性CV<5% |
| EEG信号质量 | SNR>20dB，电极阻抗<50kΩ |
| 卡匣寿命 | 每个卡匣≥60次标准疗法（30min/次） |
| 噪声 | 微流控泵工作噪声<35dBA |
| 外观 | 桌面摆件风格，非医疗器械外观 |
| 安全性 | 精油为食品级/化妆品级，无有毒挥发物 |
| 操作 | 老人单键启动，全程自动，无需操作 |

### 6. 约束条件与合规要求

- 精油原料须符合GB 22300化妆品嗅觉原料标准
- 设备本身申报为**健康管理设备**，精油卡匣申报为**卫生用品**
- 不得在宣传材料中声称治疗效果，仅可声称"辅助认知健康管理"
- EEG头带部分遵循YY/T 1195电生理传感器标准
- 精油挥发物浓度须低于OSHA工作场所暴露限值的10%

## B. 详细设计方案

### 1. 系统架构

```
[EEG头带]
     │ BLE 5.0
     ▼
[嗅觉唤醒仪主机]
     ├── 8卡匣微流控气味释放系统
     ├── 边缘AI（脑电状态分类）
     ├── 气味控制器（8路独立泵）
     └── 浓度传感器（实时校准）
          │ WiFi / BLE
          ▼
[手机APP]
     ├── 疗程管理
     ├── 嗅觉测试
     ├── 数据报告
     └── 卡匣库存管理
          ▼
[云端服务（数据分析+报告）]
```

### 2. 硬件设计方案

**2.1 微流控气味释放系统**
- 微流控泵：8路独立蠕动微泵（Bartels mp-t），流量范围0.5–10μL/min
- 气味卡匣：标准化精油卡匣（10mL容量），磁吸安装，RFID自动识别种类和剩余量
- 雾化器：超声波雾化器（频率2.4MHz），将精油微粒化为<5μm气溶胶
- 浓度传感器：MOS型气体传感器阵列（电子鼻，8通道），实时监测出风口浓度
- 风道设计：层流气道，确保气味均匀扩散，方向可调（朝向用户鼻部）

**2.2 EEG头带**
- 单通道（Fp1-A1参考），干式金电极，嵌入软硅胶头带
- 芯片：ADS1115（4通道ADC，低功耗）+ 定制前放电路
- 通信：BLE 5.0，连接主机

**2.3 主机计算**
- MCU：STM32H7（Cortex-M7，216MHz，运行DSP库）
- 气味控制固件：实时PID控制（流量闭环）
- 边缘AI：TFLite on MCU（脑电状态分类模型，<50KB）

**2.4 结构设计**
- 外观：陶瓷质感白色球形底座（直径18cm），上部出风格栅，极简日式风格
- 卡匣仓：后侧翻盖，8卡匣环形排列，LED环提示当前使用中的卡匣
- 操作：顶部一个触控按钮（开始/暂停疗程）

### 3. 软件设计方案

**3.1 主机固件**
```
├── EEG信号接收（BLE GATT）
├── 实时脑电处理（带通滤波→FFT→功率谱→状态分类）
├── 气味决策引擎（脑电状态→气味配方查找表+PID控制）
├── 微流控驱动（8路蠕动泵独立PWM控制）
├── 电子鼻监测（浓度反馈，闭环修正流量）
└── WiFi通信（与APP同步状态、上传评估数据）
```

**3.2 手机APP**
```
├── 疗程模块
│     ├── 疗程选择（记忆强化/情绪调节/嗅觉训练）
│     ├── 疗程日历（打卡记录）
│     └── 实时状态显示（当前气味、脑电状态可视化）
├── 嗅觉测试模块
│     ├── 气味辨别测试（释放→选择名称）
│     └── 嗅觉敏感度趋势图
├── 卡匣管理
│     ├── 8个卡匣剩余量显示
│     └── 低量提醒+一键补购
└── 报告模块（月度认知+嗅觉综合报告）
```

### 4. 核心算法设计

**4.1 脑电-气味闭环控制策略**

| 脑电状态 | 判断条件 | 气味配方 | 预期效果 |
|----------|----------|----------|----------|
| 认知迟缓 | α/β>2.0 | 迷迭香80%+薄荷20% | 提神激活，提升β波活动 |
| 焦虑紧张 | β高功率>基线50% | 薰衣草70%+橙花30% | 镇定，降低β波 |
| 记忆训练 | α中等，θ偏低 | 柠檬60%+茉莉40% | 记忆关联激活，提升θ波 |
| 疲劳 | 全频低功率，体动增多 | 薄荷90%+松木10% | 清醒提神 |
| 最优状态 | α/β在1.0–1.5 | 维持当前配方或停止释放 | 保持当前状态 |

**4.2 浓度闭环PID控制**
```
目标浓度 C_target（由决策引擎设定，单位ppm）
实测浓度 C_measured（电子鼻输出）

误差 e = C_target - C_measured
流量调整 Q = Kp*e + Ki*∫e dt + Kd*(de/dt)
限幅：Q ∈ [0.1, 5.0] μL/min

Kp=0.5, Ki=0.01, Kd=0.1（通过系统辨识整定）
```

**4.3 嗅觉测试评分算法**
- 释放5种随机气味（从8种中选）
- 每次释放后用户从4个选项中选择（3选项干扰+1正确）
- 分数：正确得20分，错误得0分，响应时间<10秒得额外5分
- 敏感度指数：连续3次月测均值变化率，下降>20%触发预警

### 5. 用户交互设计

**5.1 老人端（极简操作）**
```
坐舒适位置 → 戴上EEG头带 → 按顶部按钮
设备语音："开始今天的嗅觉疗法，请放松呼吸..."
（自动运行30分钟，偶尔有引导语音："请专注感受这个气味..."）
结束：语音"今日疗法完成！状态良好，继续保持~"
```

**5.2 APP界面重点**
- 主页大卡片：今日疗法状态（已完成/待完成）
- 实时显示：当前释放气味的图标+浓度条
- 脑波可视化：简化为情绪小人（表情从困倦→清醒）

### 6. 商业化路径与技术风险

**商业化路径**
- **硬件**：4,200元（含8种初始卡匣各1个）
- **耗材订阅**：卡匣补充包，129元/套（8种各1个，约2个月用量）
- **B端**：养老机构多人共享版（不含EEG，仅程序化气味释放），2,800元/台
- **临床合作**：与记忆门诊合作，作为嗅觉康复处方设备

**技术风险评估**

| 风险 | 等级 | 应对措施 |
|------|------|----------|
| 个体嗅觉差异导致效果不稳定 | 高 | 个性化嗅觉测试建立基线，定制疗法方案 |
| 微流控泵精油残留堵塞 | 高 | 每次使用后自动空气冲洗流程，卡匣设计带单向阀 |
| 电子鼻灵敏度漂移 | 中 | 每月校准提醒，云端模型更新补偿漂移 |
| 单通道EEG信号质量不稳定 | 中 | 增加信号质量指示，质量低时切换为固定疗程模式 |
| 过敏反应（精油） | 低 | 首次使用前进行皮肤测试建议，提供无过敏原配方卡匣 |
