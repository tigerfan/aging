# P01 脑波镜——脑电波认知训练头环

---

## A. 需求分析文档

### 1. 产品概述与定位

**脑波镜**是一款面向55岁以上中老年人的消费级神经反馈认知训练设备，通过非侵入性脑电波（EEG）采集、实时神经反馈与AR视觉引导，帮助用户进行早期认知功能干预，延缓轻度认知损害（MCI）进展，实现"非药物、居家化、游戏化"的脑健康管理。

**市场定位**：填补医院级EEG设备（昂贵、复杂）与普通脑电健康手环（功能单薄）之间的空白；面向有健康意识的老年人及其子女，定价区间5,000–10,000元。

### 2. 目标用户画像

| 维度 | 描述 |
|------|------|
| 核心用户 | 55–75岁、无明显认知障碍但有健忘困扰的城市中老年人 |
| 次要用户 | 轻度认知损害（MCI）患者，在家属监督下使用 |
| 购买决策者 | 子女（礼品场景）或本人（健康意识强） |
| 技术接受度 | 中低，需极简操作，步骤不超过3步 |
| 使用场景 | 早晨/睡前，每日1次，每次20–30分钟，居家沙发/床头 |

### 3. 市场痛点与机会

**痛点：**
- 中国60岁以上MCI患者超过3,800万，尚无有效药物，非药物干预需求巨大
- 现有神经反馈设备（Muse、BrainCo）面向冥想/专注力，缺乏针对老年认知衰退的专项训练
- 医院神经康复科排队难、成本高，居家方案空白
- 老年人对摄像头隐私顾虑强，不愿用视频监控类产品

**机会：**
- 神经反馈疗法（NFB）循证医学证据增多，进入主流临床视野
- 消费级干电极EEG成本已降至百元级，商业化窗口开启
- AR眼镜轻量化，老年人接受度逐步提升
- 医保对认知干预的覆盖政策收紧前的窗口期

### 4. 功能性需求（FR）

**FR-01 脑电采集**
- 至少8通道干式EEG电极，覆盖前额叶（Fp1/Fp2）、顶叶（P3/P4）、颞叶（T3/T4）
- 实时采样率≥256Hz，带通滤波0.5–45Hz
- 电极阻抗自动检测，接触不良实时提醒

**FR-02 认知评估**
- 首次使用时进行10分钟基线评估，建立个人神经基准
- 每月自动生成认知趋势报告（注意力、工作记忆、执行功能三维度）
- 与MoCA量表结果可对比导入

**FR-03 神经反馈训练**
- 基于θ/α/β波比值，实时调整AR视觉场景（如花朵绽放/凋谢）
- 提供"注意力强化""工作记忆""情绪平衡"三种训练模式
- 训练难度自适应，根据近7次成绩动态调节

**FR-04 AR引导界面**
- 配套轻量AR眼镜（重量<80g），无线连接头环
- 视觉场景不超过3种元素，字体≥24pt，色彩对比度≥4.5:1
- 可切换为纯语音引导模式（无需AR眼镜）

**FR-05 数据管理**
- 训练数据本地存储（头环内置4GB）+云端备份（加密传输）
- 家属端APP可查看训练进度、趋势图
- 支持导出PDF报告给医生

**FR-06 告警与建议**
- 连续5次训练波形异常时，推送专业就医建议
- 发现疑似癫痫样放电波形，立即告警并建议停用

### 5. 非功能性需求（NFR）

| 维度 | 指标 |
|------|------|
| 可用性 | 开机到开始训练≤3步操作 |
| 舒适性 | 佩戴重量<120g，连续使用30分钟无明显压迫感 |
| 电池续航 | 单次充电≥6次完整训练（约180分钟） |
| 连接稳定性 | BLE连接丢包率<0.1%，重连时间<3秒 |
| 数据安全 | 符合GB/T 35273个人信息安全规范，医疗数据本地优先 |
| 系统响应 | 神经反馈延迟<100ms（感知阈值以内） |
| 可靠性 | MTBF>8,000小时，电极寿命>1,000次使用 |

### 6. 约束条件与合规要求

- 作为**II类医疗器械**（神经反馈治疗仪）申报NMPA，取得注册证
- 禁忌症：癫痫患者、植入心脏起搏器者、头部金属植入物者禁用
- 电极材料：采用医用级Ag/AgCl涂层，无皮肤过敏风险
- 电气安全：符合YY 0505（医用电气设备EMC标准）、IEC 60601-1
- 隐私合规：EEG原始数据不上传云端，仅上传特征向量

## B. 详细设计方案

### 1. 系统架构

```
[EEG头环] ──BLE 5.0──► [AR眼镜 / 手机APP]
     │                         │
     │ 原始脑电信号               │ 训练指令/场景控制
     ▼                         ▼
[片上MCU+DSP] ──► [边缘AI推理] ──► [神经反馈引擎]
     │                                   │
     ▼                                   ▼
[本地Flash存储]                    [AR场景渲染]
     │
     ▼（加密同步）
[云端用户数据库] ──► [报告生成服务] ──► [家属APP / 医生端]
```

### 2. 硬件设计方案

**2.1 传感模块**
- 核心芯片：TI ADS1299（8通道、24bit ADC、专为生物电位采集设计）
- 干式EEG电极：镀金弹簧针式电极，弹力范围0.5–2N，适应发质差异
- 参考电极：耳夹式湿凝胶参考（可换洗耳垫），降低共模噪声
- 加速度计：ICM-42688（6轴IMU），用于运动伪迹检测与去除

**2.2 处理模块**
- 主控：Nordic nRF5340（双核ARM Cortex-M33，BLE 5.0）
- AI推理加速：运行TFLite Micro模型（<200KB），实时θ/α/β功率谱分类
- 本地存储：4GB NAND Flash（存储约6个月训练数据）

**2.3 通信与电源**
- 通信：BLE 5.0 + WiFi 6（通过手机中继到云端）
- 无线充电：Qi协议，5W，充电座兼具UV-C LED消毒功能
- 电池：800mAh锂电，续航约6次完整训练（共约180分钟）

**2.4 结构设计**
- 形态：环形头带式（类似耳机头梁），硅胶材质，一体注塑
- 尺寸调节：无极旋钮调节，适应头围54–62cm
- IP防护：IP22（防汗水滴落）

**2.5 配套AR眼镜**
- 方案：单目光波导，FOV≥30°，重量<70g
- 显示：MicroLED，亮度>2,000nit
- 算力：高通XR1平台（独立运行）
- 电池：内置400mAh，续航2小时

### 3. 软件设计方案

**3.1 头环固件分层架构**

| 层级 | 功能 |
|------|------|
| 驱动层 | ADS1299驱动、IMU驱动、BLE协议栈 |
| 信号处理层 | 50/60Hz陷波、带通滤波、FastICA伪迹去除 |
| 特征提取层 | FFT频谱分析、功率谱密度计算、相干性分析 |
| AI推理层 | TFLite Micro认知状态分类模型 |
| 通信层 | BLE GATT服务、AES-128加密传输 |

**3.2 AR眼镜应用（Unity轻量化，针对XR1优化）**
- 训练场景渲染引擎：3种场景（花园、星空、水墨）
- BLE数据接收与解析
- 神经反馈映射：脑波特征 → 视觉场景参数（花朵开合速度、星点密度）
- 语音TTS引导（离线，粤语/普通话可选）
- 适老UI：大字体、高对比、防眩光滤镜

**3.3 手机APP（iOS/Android）**
- 设备管理：配对、固件OTA、电极接触热力图
- 训练管理：计划制定、训练记录、成就徽章
- 报告模块：认知趋势图、月度PDF报告
- 家属互联：绑定、进度分享、异常推送通知
- 医生对接：EDF+格式数据导出

**3.4 云端服务（微服务架构）**
- 用户数据库：AES-256加密存储
- 模型更新服务：OTA推送新推理模型
- 报告生成：Python + ReportLab
- 告警服务：异常波形检测、就医建议推送

### 4. 核心算法设计

**4.1 实时频谱分析**
- Welch方法估计功率谱密度（PSD），窗长2秒，50%重叠
- 计算θ（4–8Hz）、α（8–13Hz）、β（13–30Hz）频带功率
- 关键指标：θ/α比（注意力）、α/β比（放松度）

**4.2 伪迹自动去除**
- 眼电（EOG）：FastICA，自动识别眼跳/眨眼成分并回归去除
- 肌电（EMG）：功率谱>30Hz能量突增时标记数据段
- 运动：IMU加速度>0.2g时触发数据段剔除

**4.3 认知状态分类模型**
- 架构：轻量1D-CNN（输入8通道×256采样点，参数量<50K）
- 输出：注意力等级（0–100）、工作记忆负荷（低/中/高）、疲劳度
- 训练数据：联合医院采集1,000名老年人基线数据
- 更新策略：每季度在线学习，结合用户历史数据个性化微调

**4.4 自适应难度引擎**
- 滑动窗口（最近7次）计算平均得分
- 连续3次得分>85，自动提升训练强度一级
- 连续2次得分<40，自动降低强度并发送鼓励消息

### 5. 用户交互设计

**5.1 物理交互**
- 单一按键：长按3秒开机，单击切换模式，双击暂停
- 状态指示：环形呼吸灯（绿=正常，黄=充电，红=故障）
- 电极接触质量：APP实时显示8通道接触热力图

**5.2 训练流程（端到端）**
```
佩戴头环（30秒自动检测电极）
    ↓
APP语音引导："请闭眼，保持放松，训练开始"
    ↓
AR眼镜/语音引导20分钟训练（全自动）
    ↓
训练结束：语音播报今日得分+鼓励语
    ↓
APP显示本次训练简报（得分、状态趋势、明日建议）
```

**5.3 家属端体验**
- 每周推送子女关怀消息："妈妈本周坚持训练5次，认知指数提升3%"
- 家属可代替用户设置训练提醒（老年人操作减负）

### 6. 商业化路径与技术风险

**商业化路径**
- **阶段1**（0–18个月）：II类器械注册，与三甲医院神经内科合作临床验证
- **阶段2**（18–36个月）：线上DTC + 线下药店/健康管理中心
- **阶段3**（36个月+）：与商业医保合作，纳入认知干预保险产品
- **收费模式**：硬件6,800元 + 云端订阅99元/月（含月度报告）

**技术风险评估**

| 风险 | 等级 | 应对措施 |
|------|------|----------|
| 干式电极接触质量差 | 高 | 导电凝胶辅助包，定期提醒清洁头皮 |
| EEG个体差异大，模型泛化差 | 高 | 2周个性化基线校准期 |
| AR眼镜老年人适配性差 | 中 | 提供"无眼镜模式"（纯语音+手机屏） |
| NMPA注册周期长（约3年） | 中 | 先以健康管理设备上市，同步推进注册 |
| 用户依从性低 | 中 | 游戏化激励体系、家属督促机制 |
