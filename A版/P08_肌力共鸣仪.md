# P08 肌力共鸣仪——肌电反馈居家康复手环

---

## A. 需求分析文档

### 1. 产品概述与定位

**肌力共鸣仪**是一款针对老年肌少症（Sarcopenia）的居家康复手环，通过表面肌电（sEMG）实时采集上肢肌肉活动信号，结合神经肌肉电刺激（NMES）和生物反馈游戏，为老年人提供科学化、游戏化的肌力训练方案。配合数字孪生技术构建个人肌肉模型，精准评估肌力状态并生成个性化训练处方。

**蓝海点**：肌少症是被严重低估的老龄化疾病（全球患病率超20%），现有干预手段以营养补充和传统运动为主，智能化sEMG+NMES居家设备市场几乎空白。

### 2. 目标用户画像

| 维度 | 描述 |
|------|------|
| 核心用户 | 65–80岁，有肌肉无力、日常活动困难的老年人 |
| 次要用户 | 脑卒中后上肢肌力康复患者，骨科术后康复期 |
| 场景 | 每日2次，每次20分钟，居家椅子/沙发坐姿 |
| 医疗背景 | 康复科医生建议、营养科转介 |
| 依从性风险 | 传统康复枯燥，游戏化机制是关键差异点 |

### 3. 市场痛点与机会

- 中国60岁以上肌少症患者约2,000万，跌倒、失能风险显著升高
- 康复医院资源严重不足，居家康复需要专业化工具支持
- sEMG消费级设备（Myo、Delsys）主要面向运动员和研究人员，非老年康复市场
- 电刺激设备（低频理疗仪）功能单一，缺乏个性化方案和效果评估
- 游戏化健康（Gamification Health）在老年群体中被证明可显著提升依从性

### 4. 功能性需求（FR）

**FR-01 肌电采集**
- 4通道sEMG（前臂屈肌群、伸肌群、肱二头肌、三角肌前束各1通道）
- 采样率1,000Hz，带通滤波20–500Hz，共模抑制比（CMRR）>80dB
- 皮肤电极：干式Ag/AgCl涂层，按压式快速贴合

**FR-02 肌肉状态评估**
- 最大随意收缩（MVC）测试，建立个人肌力基线
- 实时计算：肌肉激活程度（%MVC）、疲劳指标（平均频率下降率）、左右对称性
- 每月输出肌力趋势报告，与正常老年人群基准对比分位数排名

**FR-03 神经肌肉电刺激（NMES）**
- 双通道NMES输出（屈肌群/伸肌群独立控制）
- 刺激参数：频率20–80Hz可调，脉冲宽度200–400μs，强度0–30mA（用户自调）
- 模式：肌肉增强模式（高频）、疲劳恢复模式（低频）、协同训练模式（sEMG触发NMES）
- 安全限制：强度超过20mA需医生授权解锁

**FR-04 生物反馈游戏**
- 肌电信号实时控制手机游戏角色（如：握拳控制飞船方向，放松控制速度）
- 游戏场景3种：力量训练（握力方向控制）、耐力训练（保持激活水平在目标区间）、协调训练（双手肌电镜像任务）
- 难度自适应（基于近5次训练MVC%完成率）

**FR-05 数字孪生肌肉模型**
- 基于年龄、性别、BMI、sEMG特征构建个人肌肉力学模型
- 预测：当前肌力水平对日常功能任务（提重物、上楼梯）的影响
- 可视化：3D肌肉状态热力图（哪组肌肉薄弱）

**FR-06 训练处方管理**
- 首次使用：基线评估（10分钟）→ AI生成个性化4周训练处方
- 每周根据完成情况动态调整处方
- 处方可与康复医生共享（PDF格式）

### 5. 非功能性需求（NFR）

| 维度 | 指标 |
|------|------|
| sEMG信号质量 | SNR>30dB，基线漂移<5μV/s |
| NMES安全性 | 符合YY 0607医用电气设备标准，漏电流<10μA |
| 续航 | 单次充电≥4小时（含NMES输出），USB-C充电 |
| 重量 | 手环重量<120g（单只） |
| 舒适性 | 佩戴30分钟无皮肤刺激，电极可洗 |
| 游戏响应延迟 | sEMG控制游戏延迟<50ms |

### 6. 约束条件与合规要求

- NMES功能须作为**II类医疗器械**（神经肌肉电刺激器）申报
- 禁忌症：植入心脏起搏器、皮肤破损区域、活动性深静脉血栓禁用
- 电气安全：IEC 60601-1-2，Type BF（体表浮地应用）
- 儿童锁：NMES强度调节需二次确认，防止老人误操作到最大强度

## B. 详细设计方案

### 1. 系统架构

```
[肌力共鸣仪（双腕/前臂佩戴）]
     ├── 4通道sEMG前放+ADC模块
     ├── 双通道NMES输出模块（恒流源）
     ├── 边缘MCU（实时信号处理+BLE）
     └── 触控显示（1.3寸圆形彩屏）
          │ BLE 5.0
          ▼
[手机APP]
     ├── 生物反馈游戏（Unity/Godot）
     ├── 训练管理
     ├── 数字孪生可视化
     └── 医生报告
          ▼
[云端服务]
     ├── 个性化处方生成（AI模型）
     ├── 人群基准数据库
     └── 数字孪生建模服务
```

### 2. 硬件设计方案

**2.1 sEMG采集模块**
- 前放芯片：TI INA333（3-op仪表放大器，增益×100，CMRR>100dB）
- ADC：ADS1298（8通道，24bit，1kSPS，低噪声生物信号ADC）
- 电极：四对Ag/AgCl干电极，间距20mm，按压式弹簧针

**2.2 NMES输出模块**
- 恒流源：LM334改进电路，0–30mA恒流输出
- 隔离：光电隔离+变压器隔离（双重隔离保证安全）
- 波形：双相方波（消除直流成分，避免皮肤灼伤）
- 安全保护：过流保护（>35mA立即截断）、电极脱落检测（阻抗>500kΩ报警停机）

**2.3 主控与显示**
- MCU：Nordic nRF5340（低功耗，BLE 5.0，运行sEMG实时处理）
- 显示：1.3英寸圆形TFT（240×240，SPI接口），显示实时肌电波形和训练进度
- 触控：电容式单点触控（大按钮操作）

**2.4 结构设计**
- 手环主体：医用硅胶+ABS外壳，腕带宽45mm，覆盖前臂远端1/3
- 电极阵列：内侧均匀分布4对电极，导电硅胶垫缓冲
- 防水：IPX4（防汗液，可用湿布清洁）

### 3. 软件设计方案

**3.1 手环固件**
```
├── ADC驱动（DMA采集，1kHz）
├── 实时信号处理
│     ├── 50Hz工频陷波（IIR）
│     ├── 带通滤波（20-500Hz，Butterworth 4阶）
│     ├── RMS包络提取（100ms窗口）
│     ├── 中值频率计算（疲劳指标）
│     └── MVC百分比归一化
├── NMES控制器（PWM波形生成，DMA输出）
├── BLE GATT服务（实时数据流 + 配置控制）
└── 显示UI（实时波形、进度条、训练状态）
```

**3.2 手机游戏引擎（Unity）**
```
├── BLE数据接收插件（实时sEMG流）
├── 游戏逻辑
│     ├── 场景1：太空飞船（握力控制方向）
│     ├── 场景2：钢丝行者（保持激活目标区间）
│     └── 场景3：镜像挑战（双手对称肌电同步）
├── 难度自适应引擎
└── 成就系统（徽章、积分、排行榜）
```

**3.3 数字孪生建模**
- 输入：年龄、性别、BMI、每日sEMG最大值、肌肉疲劳曲线
- 模型：参数化肌肉力学模型（Hill肌肉模型变体），云端运行
- 输出：预测最大抓握力（kg）、预测上肢功能任务耐力
- 更新：每次训练后上传数据，模型增量更新

### 4. 核心算法设计

**4.1 sEMG疲劳实时检测**
```python
# 中值频率（MDF）下降率作为疲劳指标
def compute_fatigue_index(emg_signal, window_s=1.0, fs=1000):
    windows = sliding_window(emg_signal, window_s, overlap=0.5)
    mdf_series = [median_frequency(fft(w), fs) for w in windows]
    # 线性回归斜率：负值越大，疲劳越快
    slope = linear_regression_slope(mdf_series)
    fatigue_index = max(0, -slope / baseline_slope * 100)  # 归一化0-100
    return fatigue_index
```

**4.2 肌力基线自动校准（MVC测试）**
- 引导用户做最大努力等长收缩（5秒×3次，间歇2分钟）
- 取3次RMS峰值的最大值为MVC基准
- 每4周重新测试更新基准（追踪肌力改善）

**4.3 NMES-sEMG协同模式**
- 检测到用户主动肌肉收缩意图（sEMG>15%MVC）时，触发NMES辅助输出
- NMES强度随主动收缩强度自适应调整（主动越强，电刺激越弱）
- 原理：模拟正常神经肌肉协调，促进神经可塑性恢复

### 5. 用户交互设计

**5.1 老人端操作流程**
```
佩戴手环（电极自动贴合）→ 按下圆形主按钮
→ 屏幕显示"信号检测中..."（3秒）→ "信号良好，开始训练！"
→ 打开手机APP，选择今日游戏场景
→ 按照屏幕提示做动作（游戏自动引导），20分钟
→ 训练结束，屏幕显示今日肌力评分
```

**5.2 可视化反馈**
- 手环屏幕：实时RMS波形 + 目标区间（绿色区间，保持在区间内得分）
- 手机游戏：肌电直接控制游戏角色，即时视觉反馈
- 每周推送：肌力热力图（4个肌肉组的改善雷达图）

**技术风险评估**

| 风险 | 等级 | 应对措施 |
|------|------|----------|
| 老年人皮肤阻抗高，sEMG信号质量差 | 高 | 导电凝胶辅助垫，皮肤预处理酒精棉片 |
