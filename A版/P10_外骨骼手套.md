# P10 外骨骼手套——软体康复外骨骼手套

---

## A. 需求分析文档

### 1. 产品概述与定位

**外骨骼手套**是一款面向脑卒中后遗症患者手部功能康复的软体气动外骨骼设备，通过柔性气囊驱动手指被动/主动辅助运动，结合表面肌电信号检测患者运动意图，实现"意图感知-辅助执行"的闭环康复训练，配合游戏化任务（如虚拟抓球、捏物）促进神经可塑性重塑，帮助患者在家中持续进行专业级手部康复训练。

**蓝海点**：中国脑卒中存活患者超1,500万，手部功能障碍发生率约70%，居家康复设备极度匮乏；现有医院级手部外骨骼（Gloreha等）价格超30万元，消费化空间巨大。

### 2. 目标用户画像

| 维度 | 描述 |
|------|------|
| 核心用户 | 脑卒中后3个月–2年，手部功能障碍（Brunnstrom分期III–V期）患者 |
| 年龄范围 | 55–80岁（脑卒中高发年龄段） |
| 使用场景 | 居家，每日2次，每次30分钟，坐姿桌前 |
| 照护人 | 家属或护理员辅助佩戴，设备自动运行 |
| 医疗背景 | 康复科、神经内科医生制定康复方案 |

### 3. 市场痛点与机会

- 脑卒中后手部康复黄金期（发病后6个月内）常因医疗资源不足而错过
- 居家康复缺乏专业设备引导，效果差且依从性低
- 医院级外骨骼设备价格昂贵（20–50万元），仅少数三甲医院配备
- 软体外骨骼（气动驱动）成本已可降至千元级，商业化窗口开启
- 神经可塑性研究证实：高强度、高重复次数（每日>200次手指运动）是康复关键

### 4. 功能性需求（FR）

**FR-01 手部被动运动**
- 5根手指气囊独立驱动（可单指、多指、全手控制）
- 运动模式：全指屈曲、全指伸展、侧捏（拇指-食指）、抓握
- 运动速度可调（慢/中/快三档），适应不同康复阶段
- 单次运动循环时长2–8秒可调

**FR-02 主动辅助训练（sEMG意图检测）**
- 前臂sEMG检测患者主动收缩意图（屈肌/伸肌各1通道）
- 检测到主动意图（阈值>15%MVC）时，触发对应气囊辅助运动
- 辅助力度自适应：患者主动力越大，气囊辅助越小（中枢-外周协同）

**FR-03 游戏化任务训练**
- 手机/平板显示虚拟场景，手套传感器（弯曲传感器）检测手指实际姿态
- 游戏任务：
  - 虚拟抓球（全掌抓握）
  - 捏豆子（侧捏精细动作）
  - 弹钢琴（单指独立控制）
  - 水果切切（伸展动作）
- 5个难度级别，自适应调整

**FR-04 功能评估**
- 手部功能量表数字化版：Fugl-Meyer上肢量表简化版（FMA-UE），每2周测评
- 抓握力评估（气囊反压力估算）
- 手指关节活动度（ROM）实时显示
- 训练次数统计（每日手指运动总次数）

**FR-05 康复方案管理**
- 医生端：制定个性化康复处方（训练模式、强度、每日目标次数）
- 患者端：按方案自动执行，完成打卡
- 进度共享：每周向主治医生推送训练数据和评估结果

**FR-06 远程监护**
- 家属APP实时查看训练状态
- 训练完成后推送摘要
- 设备异常（气囊压力异常、传感器脱落）实时告警

### 5. 非功能性需求（NFR）

| 维度 | 指标 |
|------|------|
| 气囊压力控制精度 | ±5kPa |
| sEMG意图检测延迟 | <100ms（保证自然感） |
| 安全压力上限 | 单指气囊≤80kPa（超压自动泄气） |
| 重量 | 手套主体<200g（含气管），气泵主机<500g |
| 适应性 | 一款手套适应手掌周长18–24cm（S/M/L三码） |
| 防污染 | 内衬可拆卸清洗，外层酒精擦拭消毒 |
| 续航 | 主机电池≥2小时连续训练 |

### 6. 约束条件与合规要求

- 作为**II类医疗器械**（运动功能障碍辅助治疗设备）申报NMPA
- 禁忌症：手部严重痉挛（改良Ashworth量表>3级）、手部皮肤破损、骨折未愈合
- 气压安全：每根气囊独立过压保护阀，气泵输出最大压力硬件限制≤150kPa
- 电气安全：IEC 60601-1，Type BF设备
- 儿童锁：sEMG阈值和气囊压力修改须输入医生授权码

## B. 详细设计方案

### 1. 系统架构

```
[软体外骨骼手套]
     ├── 5路硅胶气囊（手指背侧）
     ├── 5路弯曲传感器（手指姿态）
     └── sEMG电极片×4（前臂，检测意图）
          │ 气管×5（φ4mm硅胶管）
          ▼
[气泵主机]
     ├── 5路微型气泵（独立控制）
     ├── 5路电磁阀（充/放气切换）
     ├── 压力传感器×5（实时监测）
     ├── MCU（控制+BLE）
     └── 锂电池（3,000mAh）
          │ BLE 5.0
          ▼
[手机/平板APP]
     ├── 游戏化训练界面
     ├── 评估模块
     └── 医生/家属联结
```

### 2. 硬件设计方案

**2.1 软体气囊手套**
- 气囊材料：Dragon Skin系列硅橡胶（30A硬度），壁厚1.5mm
- 气囊结构：波纹管式（充气时弯曲力臂使手指屈曲），单指最大屈曲力矩约0.5Nm
- 手套基材：莱卡弹性织物，透气性好，内衬抗菌银纤维
- 弯曲传感器：导电橡胶弯曲传感器（阻值随弯曲角度线性变化，±2°精度）

**2.2 气泵主机**
- 微型气泵：5个THOMAS 1102CDC微型隔膜泵（最大压力200kPa，流量1L/min）
- 电磁阀：SMC S070系列微型电磁阀（5路，响应时间<15ms）
- 压力传感器：Honeywell MPRLS（24bit，±0.25%精度）×5
- 连接器：医用级快插接头（手套与主机快速连接/断开，单手操作）

**2.3 sEMG模块**
- 4通道：屈肌群2通道、伸肌群2通道，采用ADS1298芯片
- 电极：Ag/AgCl干电极，前臂腕横纹近端5–10cm处

**2.4 主控**
- MCU：STM32H7（控制气泵PWM + sEMG采集 + BLE通信）
- BLE：Nordic nRF52840模组（低功耗BLE 5.0）
- 电源：18650锂电×2（7.4V/5Ah），USB-C PD 20W快充，约2小时充满

### 3. 软件设计方案

**3.1 主机固件**
```
├── sEMG信号处理（带通20-500Hz，RMS提取，意图分类）
├── 气囊控制器
│     ├── 模式执行（被动模式：定时序列；主动模式：sEMG触发）
│     ├── PID压力控制（目标压力±5kPa）
│     └── 安全监控（超压/欠压/泄漏检测）

├── BLE数据流（实时推送sEMG+姿态+压力数据到APP）

```

**3.2 手机游戏APP**
```
├── 实时手指姿态渲染（3D手模型，与真实手指同步）
├── 游戏逻辑引擎（Unity）
│     ├── 场景1：抓球（手指闭合检测）
│     ├── 场景2：捏豆子（拇指-食指夹角<20°判定）
│     ├── 场景3：弹钢琴（单指独立弯曲检测）
│     └── 场景4：伸展挑战（最大伸展角度比赛）

```

### 4. 核心算法设计

**4.1 运动意图检测**
```python
def detect_motion_intention(emg_flexor, emg_extensor, mvc_baseline):
    # 计算MVC百分比
    flex_pct = rms(emg_flexor) / mvc_baseline['flexor']
    ext_pct = rms(emg_extensor) / mvc_baseline['extensor']
    
    if flex_pct > 0.15:
        return 'FLEXION', flex_pct  # 屈曲意图
    elif ext_pct > 0.15:
        return 'EXTENSION', ext_pct  # 伸展意图
    else:
        return 'REST', 0
```

**4.2 辅助力度自适应**
- 原理：主动收缩越强，外骨骼辅助越少（鼓励患者主动参与）
- 计算：`assist_pressure = MAX_PRESSURE × (1 - patient_effort_ratio)`
- 初期（Brunnstrom III）：assist_ratio=90%；后期（V期）：assist_ratio=30%

**4.3 痉挛检测与保护**
- 实时监测气囊反压力：若充气过程中压力上升但位移传感器无变化，判断为肌肉痉挛
- 痉挛检测后：立即暂停当前动作，缓慢泄气，语音提示"检测到肌肉紧张，暂停训练，请放松"
- 记录痉挛事件，推送给主治医生

### 5. 用户交互设计

**5.1 佩戴与启动流程**
```
家属/患者将手套戴上（魔术贴固定腕部）
→ 连接气管（快插接头，一听"咔哒"声确认）
→ 气泵自动检测气密性（30秒）→ 检测sEMG信号（5秒）
→ APP显示"设备就绪，开始今日训练？" → 确认
→ 训练开始（被动/主动辅助模式按处方执行）
```

**5.2 游戏界面（适老化）**
- 游戏目标简单清晰：一个大目标物（球/豆子），单次任务<10秒
- 成功动效：彩色烟花+音效（正强化）
- 失败：温和提示"再试一次！"（不使用"失败"字样）
- 进度条：显示今日已完成次数/目标次数（200次/日）

### 6. 商业化路径与技术风险

**商业化路径**
- **定价**：4,600元（含气泵主机+标准手套）；替换内衬耗材399元/套（每3个月更换）
- **服务**：康复处方订阅，199元/月（含远程医生随访、方案调整）
- **渠道**：神经内科/康复科医院推荐（处方销售），DRG支付改革背景下促进院外康复需求
- **B端**：康复医院训练室采购，护理站批量使用

**技术风险评估**

| 风险 | 等级 | 应对措施 |
|------|------|----------|
| 手部痉挛导致气囊无法展开 | 高 | 痉挛实时检测+自动停止，Ashworth>3级禁用 |
| 气管连接处漏气 | 中 | 快插接头气密性测试，出厂前100%气密检验 |
| sEMG电极脱落（老年人皮肤松弛） | 中 | 自粘式电极片+弹性臂带固定，信号质量实时监测 |
| 老年人手部差异大（关节僵硬、变形） | 中 | 气囊充气速度和压力上限个性化设置，柔性材料适应变形 |
| NMPA注册周期 | 中 | 同步开展临床试验，预计3年注册周期，先以"体感游戏设备"上市 |
