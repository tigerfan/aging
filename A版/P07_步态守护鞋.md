# P07 步态守护鞋——智能步态分析防跌倒鞋

---

## A. 需求分析文档

### 1. 产品概述与定位

**步态守护鞋**是一款将IMU传感器阵列、边缘AI步态分析和主动气囊保护系统集成于普通鞋底的智能防跌倒鞋。通过实时分析行走步态特征，在检测到跌倒风险姿态的瞬间（约80ms预判窗口），触发鞋底气囊充气，降低跌倒冲击伤害；同时提供步态健康评估和跌倒预防训练建议。

**核心创新**：从"跌倒后报警"（被动）到"跌倒前防护"（主动），填补被动防跌倒设备与主动干预之间的空白。

### 2. 目标用户画像

| 维度 | 描述 |
|------|------|
| 核心用户 | 65–80岁，有跌倒史或跌倒高风险的老年人（步态不稳、骨质疏松） |
| 次要用户 | 帕金森病、脑卒中后遗症患者，步态异常群体 |
| 场景 | 日常室内外行走，外出购物、公园散步 |
| 家属关注点 | 父母独居安全，跌倒是最大担忧之一 |
| 医疗需求 | 骨科/神经科医生开具使用建议 |

### 3. 市场痛点与机会

- 中国每年老年人跌倒致死超过40万例，跌倒是65岁以上老年人意外死亡首因
- 现有防跌倒产品以被动检测（手环报警）为主，无法预防跌倒本身
- 髋关节骨折（跌倒主要后果）平均住院费>5万元，一次跌倒可能彻底改变生活质量
- 智能鞋市场（运动监测为主）未涉足老年医疗防护场景
- 气囊技术已成熟（摩托车气囊背心、汽车安全气囊），微型化降本可期

### 4. 功能性需求（FR）

**FR-01 步态实时监测**
- 鞋底内置6轴IMU阵列（3个IMU，前掌/中掌/后跟各一个），采样率200Hz
- 足底压力传感器阵列（8×8柔性压力矩阵），检测压力分布
- 实时计算步态参数：步长、步频、支撑相/摆动相比例、足跟着地模式、步态对称性

**FR-02 跌倒风险预判**
- 基于步态特征，实时计算跌倒风险评分（0–100）
- 风险评分>70时，APP发出预警（振动+语音："请注意脚下安全"）
- 识别高风险姿态：重心过偏、绊脚步态、碎步（帕金森特征步态）

**FR-03 主动气囊保护**
- 内置2个微型气囊（踝关节两侧），平时折叠存储
- 检测到跌倒姿态（重心急剧偏移，速度阈值触发），80ms内气囊充气
- 气囊充气后持续2秒，自动放气，不妨碍起身
- 假阳性控制：算法优化误触发率<5次/1000小时使用

**FR-04 跌倒检测与报警**
- 若跌倒已发生（落地冲击检测），立即推送家属告警（GPS位置+时间）
- 支持SOS一键求助（长按鞋面隐藏按钮）
- 与手机联动，跌倒后30秒未响应自动拨打紧急联系人

**FR-05 步态健康评估**
- 每日步行数据自动记录（步数、步频趋势、步态不对称性变化）
- 每月生成步态健康报告，与历史基线对比
- 步态异常（如突然出现碎步、步幅减小）时，推送神经科就诊建议

**FR-06 个性化训练建议**
- 基于步态分析，生成针对性强化训练方案（如：提踵练习改善推离相）
- 配合手机APP展示训练动作视频（3D动画，适老化设计）
- 训练完成打卡，记录步态改善轨迹

### 5. 非功能性需求（NFR）

| 维度 | 指标 |
|------|------|
| 响应时间 | 气囊触发延迟<80ms（人体跌倒保护窗口内） |
| 气囊可靠性 | 触发成功率>99.5%，使用寿命≥500次充放气 |
| 续航 | 单次充电续航≥8小时连续行走，Qi无线充电 |
| 重量 | 单只鞋总重<400g（含电池和气囊） |
| 防水 | IP67（可涉水，雨天使用） |
| 舒适性 | 与普通运动鞋感觉相当，日常步行无明显不适 |
| 误触发率 | 气囊误触发<5次/1000小时 |

### 6. 约束条件与合规要求

- 气囊系统需作为**II类医疗器械**（辅助防护类）申报NMPA
- 气囊充气介质：无毒惰性气体（氮气），不含有害物质
- 电池安全：符合GB 31241锂离子电池安全标准
- 无障碍适配：鞋型遵循老年人足部特征（宽楦头、低跟、防滑大底）
- 气囊压力限制：充气压力<0.5bar，不对踝关节造成额外压力

## B. 详细设计方案

### 1. 系统架构

```
[智能鞋（左右各一）]
     ├── 6轴IMU×3（前掌/中掌/后跟）
     ├── 柔性压力矩阵（8×8）
     ├── 微型气囊×2（踝部两侧）
     ├── CO2/N2气体发生器
     ├── 边缘MCU（实时推理，步态分析）
     └── BLE 5.0 + GPS模块
          │ BLE
          ▼
[手机APP]
     ├── 步态实时数据
     ├── 跌倒告警接收
     ├── 健康报告
     └── 家属关联
          ▼
[云端分析服务]
     ├── 步态趋势分析
     ├── 跌倒事件数据库
     └── 个性化训练建议生成
```

### 2. 硬件设计方案

**2.1 传感器系统**
- IMU：InvenSense ICM-42688（6轴，±16g加速度，±2000°/s陀螺仪，采样率200Hz）
- 压力矩阵：64点压阻式柔性传感器，测量范围0–700kPa，分辨率5kPa
- 传感器分布：前掌1个IMU+32点压力，中掌1个IMU，后跟1个IMU+32点压力

**2.2 气囊系统**
- 气囊材料：热塑性聚氨酯（TPU），厚度0.3mm，展开体积约150mL
- 气体发生器：微型高压氮气储罐（4MPa），体积<20mL，单次充气量足够150mL气囊
- 触发机构：电磁阀（响应时间<10ms），NTC温控保护
- 充放气：自动放气阀（压力>0.4bar时自动泄压）

**2.3 计算与通信**
- 主控：Nordic nRF5340（双核，BLE 5.0，低功耗）
- AI推理：TFLite Micro（步态分类+跌倒风险模型，运行于M33核）
- GPS：u-blox M8N（定位精度<3m，低功耗模式）
- 电池：每只鞋内置800mAh锂电（鞋跟夹层），无线充电接收线圈

**2.4 结构集成**
- 鞋底夹层：3D打印TPU蜂窝结构，容纳传感器和电路板
- 气囊安装：踝关节两侧弹性口袋，日常折叠厚度<3mm
- 防水：整体防水处理，电子元件独立密封舱（IP67）
- 外观：与标准老年运动鞋无明显区别，宽楦头，Velcro魔术贴固定

### 3. 软件设计方案

**3.1 鞋内固件**
```
├── 传感驱动层：IMU DMA采集、压力矩阵扫描（500Hz）
├── 实时步态分析
│     ├── 步态周期检测（基于后跟着地-离地事件）
│     ├── 步态特征向量提取（步长/步频/对称性/压力分布）
│     └── 跌倒风险模型推理（TFLite Micro，<5ms推理时间）
├── 气囊控制器
│     ├── 跌倒姿态检测（多条件阈值+模型双重确认）
│     └── 触发逻辑（两只鞋通信同步，防止单只误触）
├── 通信层：BLE GATT（与手机实时同步）+ GPS（定位上报）
└── 电源管理：运动检测模式（高频采样）vs待机模式自动切换
```

**3.2 手机APP**
```
├── 实时步态仪表盘（步频、对称性实时波形）
├── 跌倒告警通知（声音+震动，家属同步）
├── 日/周/月步态报告
├── 训练模块（针对步态问题的专项练习视频）
└── 设备管理（电量、气囊状态、固件更新）
```

### 4. 核心算法设计

**4.1 跌倒风险实时评估模型**
- 架构：时序TCN（Temporal Convolutional Network），输入过去2秒数据（400帧×15特征）
- 特征：3轴加速度×3个IMU + 3轴陀螺仪×3 + 压力重心坐标
- 输出：跌倒风险概率（0–1），每50ms更新
- 在MCU上量化为INT8，推理时间<5ms

**4.2 跌倒动作识别（触发气囊）**
```
触发条件（AND逻辑）：
1. 跌倡风险概率 > 0.85
2. 合加速度 > 2.5g（重心急剧偏移）
3. 仰角变化速率 > 200°/s（快速倾倒）
4. 足底压力中心偏移 > 40mm/0.1s

触发后：
1. 立即点火气囊（10ms内）
2. 记录事件（传感器数据存档）
3. 500ms后判断是否跌倒成功（加速度峰值）
4. 若跌倒成功：触发SOS流程
```

**4.3 步态异常分类器**
- 训练数据：500名老年受试者（含帕金森、脑卒中后遗症），共2,000小时步态数据
- 检测的步态异常类型：碎步（帕金森）、拖步、不对称步、鸭步（外八）、踮脚步
- 输出：异常类别 + 严重程度（0–3级）+ 建议干预措施

### 5. 用户交互设计

**5.1 老人端操作**
- 穿上鞋：自动开机（鞋垫压力触发），无需任何操作
- 脱鞋：自动进入休眠，充电时提示灯闪烁
- SOS呼救：长按鞋侧隐藏按钮3秒（大面积防误触）
- 电量提示：鞋跟LED灯（绿/黄/红对应电量高/中/低）

**5.2 家属APP体验**
- 首页：父母实时位置地图 + 今日步行状态
- 告警推送：跌倒事件带GPS位置，一键拨打老人电话
- 周报：步态健康趋势，"本周步态对称性提升5%"

**5.3 医生端（可选）**
- 步态报告PDF，符合临床步态分析格式（包含时空参数、压力分布热力图）
- 可作为骨科/神经科随诊参考依据

### 6. 商业化路径与技术风险

**商业化路径**
- **定价**：3,800元/双（含气囊系统）+ 入门款无气囊版1,200元
- **气囊更换**：每次触发后需更换气体发生器，199元/个（含邮寄服务）
- **渠道**：骨科/神经科医院推荐、药店、电商
- **医保**：积极争取工伤保险和商业意外险覆盖
- **养老机构**：批量采购无气囊监测版，纳入安全护理标准

**技术风险评估**

| 风险 | 等级 | 应对措施 |
|------|------|----------|
| 气囊误触发（正常坐下/弯腰） | 高 | 双重确认逻辑，两只鞋通信联动，降低单只误判 |
| 跌倒前预判窗口<80ms，气囊来不及展开 | 高 | 提前进入"高警觉模式"（风险>50%时预压缩气囊弹簧） |
| 鞋内电子元件寿命（洗鞋问题） | 中 | 电子鞋垫可拆卸插拔设计，鞋本身可手洗 |
| 老年人足型差异大，舒适度问题 | 中 | 提供4款楦型（窄/标准/宽/特宽），3D足型扫描定制服务 |
| NMPA II类器械注册周期长 | 中 | 先以运动健康设备上市（无气囊版），同步推进注册 |
