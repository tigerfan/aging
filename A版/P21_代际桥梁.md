# P21 代际桥梁——多代际情感连接智能相框

---

## A. 需求分析文档

### 1. 产品概述与定位

**代际桥梁**是一款专为老年人与远距家庭成员保持情感连接设计的智能相框，突破传统视频通话的"预约式"模式，实现家庭成员的"持续在场感"。家人可随时发送照片、短视频、涂鸦留言到相框，相框自动播放展示；老人无需操作即可感受到家人动态；子女通过APP可看到老人的生活状态（在家/外出/休息），实现低打扰的双向情感连接。

**蓝海点**：现有智能相框（Amazon Echo Show等）仍需老人主动操作；亚马逊/谷歌产品无深度中文本地化；融入健康状态感知的"无操作家庭连接"产品市场空白。

### 2. 目标用户画像

| 维度 | 描述 |
|------|------|
| 老年用户 | 65–85岁，独居或空巢老人，子女不在同城 |
| 家属用户 | 25–55岁，在外工作的子女，关心父母但联系频率不足 |
| 使用场景 | 相框放置于客厅/餐厅最显眼处，全天开机 |
| 情感需求 | 老人需要感受到被惦记；子女需要低成本表达关心 |
| 技术门槛 | 老人端：零操作（相框自动运行）；子女端：如发朋友圈一样简单 |

### 3. 市场痛点与机会

- 中国空巢老人超1亿，"情感孤独"是最严重的老龄化问题之一
- 视频通话（微信）：需要老人接听操作，失聪/视力障碍老人更加困难
- 普通数码相框：单向展示，无互动，无情感温度
- 研究证实：频繁的情感连接信号（即使是简短的）能显著降低老年人孤独感
- 子女的"连接焦虑"（担心父母但不知道状态）是强烈的购买动机

### 4. 功能性需求（FR）

**FR-01 无操作内容接收**
- 子女APP发送：照片、短视频（<60秒）、语音留言、涂鸦手写（"晚饭时间到啦~"）、天气提醒
- 相框自动接收并播放，无需老人任何操作
- 播放时伴随轻柔提示音（新消息到达），音量可调
- 内容按时间轴轮播（最近7天内容循环展示）

**FR-02 生活状态感知（老人端）**
- 内置PIR被动红外传感器：检测老人是否在家、活动规律
- 毫米波雷达（可选）：非摄像头方式，检测客厅/卧室是否有人
- 状态推送至子女APP：
  - 早晨8点前后老人通常经过相框前→"早上好，爸爸今天起床了"
  - 超过12小时无活动检测→"妈妈今天活动较少，可以关心一下"
- **不使用摄像头**（保护老人隐私，消除安全顾虑）

**FR-03 双向互动（简化操作）**
- 老人端：相框下方3个实体按键（"❤️喜欢"、"📞想聊聊"、"🙏谢谢"）
- 按下按键 → 推送对应表情到子女APP（零输入操作）
- "想聊聊"按键 → 自动向子女发送视频通话邀请（子女接受后相框屏幕即通话界面）
- 一键视频通话：子女APP发起 → 相框自动接通（无需老人操作接听）

**FR-04 智能相册管理**
- 家庭相册自动同步：子女手机相册授权照片自动推送（每日精选1–3张）
- AI策划播放顺序：节假日自动播放相关照片（春节播放团圆照，生日播放生日照）
- 家族群内容聚合：孙辈的画、学校成绩、运动会视频等一键发送

**FR-05 个性化提醒**
- 子女可定时发送关心提醒（"妈，今天天冷，多穿件衣服"显示在相框）
- 自动节日提醒（老人生日、结婚纪念日相框自动显示祝福）
- 用药提醒（与用药管家P16联动，或独立设置简单提醒）

### 5. 非功能性需求（NFR）

| 维度 | 指标 |
|------|------|
| 显示 | 10英寸IPS屏，1920×1200，色彩鲜艳（老年人对色彩对比度需求高） |
| 响应速度 | 子女发送消息到相框显示<3秒 |
| 续航 | 持续联网显示（交流电供电），锂电备用4小时（停电） |
| 隐私 | 无摄像头设计，PIR/毫米波不记录任何个人识别数据 |
| 网络 | WiFi 6，离线时本地缓存内容继续播放 |
| 音量 | 扬声器最大80dB（适合听力轻度下降老人） |

### 6. 约束条件与合规要求

- PIR/雷达数据：仅本地处理，不上传原始传感数据
- 一键接通视频通话须有明确告知，防止误触被偷看
- 内容存储：家庭内容仅在家庭成员账户间流转，不被第三方使用
- 内容审核：发送内容须通过基本内容过滤（防止误传敏感内容给老人）

## B. 详细设计方案

### 1. 系统架构

```
[代际桥梁相框（客厅）]
     ├── 10英寸IPS触控屏
     ├── PIR传感器（存在检测）
     ├── 可选毫米波雷达（精细活动检测）
     ├── 3个实体快捷按键
     ├── 5W扬声器
     ├── SBC主控（Rockchip RK3566）
     └── WiFi 6 + 以太网
          │ HTTPS/MQTT
          ▼
[云端消息路由]
     ├── 实时消息推送（MQTT broker）
     ├── 媒体存储（家庭相册CDN）
     └── 状态感知数据处理
          ▼
[子女APP（iOS/Android）]
     ├── 照片/视频/涂鸦发送
     ├── 老人状态感知面板
     └── 视频通话发起
```

### 2. 硬件设计方案

**2.1 显示与计算**
- SoC：Rockchip RK3566（四核A55，支持4K解码，适合数字相框场景）
- 屏幕：10.1英寸IPS（1920×1200，300nit，色域sRGB 95%）
- 音频：2×5W全频扬声器，Dolby Audio音效处理（人声清晰度优先）

**2.2 传感器**
- PIR：AM312微型被动红外传感器，探测范围5m/120°
- 毫米波雷达（可选升级版）：60GHz FMCW雷达（德州仪器IWR6843，房间级存在检测，无隐私问题）

**2.3 实体按键**
- 3个直径30mm带背光硅胶按键（❤️/📞/🙏），触感明显
- 背光颜色：红色心形/蓝色电话/黄色双手（直觉化图标）

**2.4 外观结构**
- 仿实木边框（ABS+木纹贴膜），多色可选（原木/深胡桃/白桦）
- 立式支架（可调角度）+ 壁挂安装（附挂件）
- 尺寸：约260mm×170mm×20mm（10英寸屏比例）

### 3. 软件设计方案

**3.1 相框系统（Android AOSP定制）**
```
启动时：全屏相册轮播（家庭照片+留言）
新消息到达（MQTT推送）：
    轻提示音 → 切换到新消息展示（15秒）→ 返回轮播
    
早晨模式（6:00–9:00）：
    自动增大字体和亮度
    显示当日天气+用药提醒
    
夜间模式（22:00–7:00）：
    屏幕亮度降至最低（或关闭）
    仅新消息到达时短暂亮起
    
视频通话接入：
    子女发起 → 相框自动进入视频通话界面（无需老人操作）
    通话结束 → 自动返回相册轮播
```

**3.2 子女APP（React Native）**
- 发送界面：类微信朋友圈，照片/视频/文字/涂鸦一键发送
- 老人状态面板：今日活动时间轴（早起时间/午休/晚间活动）
- 家庭广播：一条消息发给所有老人（多个老人家庭）
- 快捷关心包：预设12套节日/场景祝福（春节/下雨天/周末），一键发送

### 4. 核心算法设计

**4.1 活动规律学习**
```python
def learn_activity_pattern(pir_events, days=30):
    # 学习老人的日常活动规律（起床时间/午休时间/睡眠时间）
    hourly_activity = defaultdict(list)
    for event in pir_events:
        hour = event.timestamp.hour
        hourly_activity[hour].append(1)
    
    normal_pattern = {h: mean(v) for h, v in hourly_activity.items()}
    return normal_pattern

def detect_anomaly(today_events, normal_pattern, threshold=0.3):
    # 今日活动与历史规律对比
    today_hourly = count_by_hour(today_events)
    for hour in range(7, 22):  # 白天时段
        if today_hourly[hour] < normal_pattern[hour] * threshold:
            if hour > 12:  # 超过中午仍无活动
                return f"{hour}时前活动较少，建议关心"
    return None
```

**4.2 AI照片策划（播放顺序优化）**
- 优先展示最近3天新照片（新鲜感）
- 每日随机选取3张"记忆重温"（1年前、5年前的同月份照片）
- 节日期间：自动检测日历，提升相关主题照片权重（过年=团圆合照优先）

### 5. 用户交互设计

**老人端（极简零操作）**
- 相框默认全屏展示，无任何图标或按钮界面
- 实体按键有物理凸感和图案，盲触可操作
- 收到新消息：温柔提示音+画面切换，自动展示

**子女端（流畅体验）**
- 主界面：老人状态卡片（在家/活跃/休息/今日发送N条）
- 发送入口：大"+"按钮，照片/视频/涂鸦/语音4个选项
- 涂鸦功能：手写"爸，吃饭啦~"并配插图，相框显示手写体（情感温度高）

### 6. 商业化路径与技术风险

**商业化路径**
- **硬件**：标准版（PIR）898元；高级版（毫米波雷达）1,280元
- **订阅**：免费基础版（50张/月存储）/ 39元/年（无限存储+节日智能策划）
- **礼品定位**：子女孝心礼物，节日（父亲节/母亲节/重阳节）销售峰值
- **渠道**：电商为主，社区卫生中心赠品合作，社区超市礼品专区

**技术风险评估**

| 风险 | 等级 | 应对措施 |
|------|------|----------|
| WiFi配置对老人太复杂 | 高 | 子女帮助首次配置，支持蓝牙快速配网（子女手机扫描完成） |
| 老人不理解自动接通视频 | 中 | 首次使用需明确授权，通话前相框显示"XX在联系您"3秒 |
| 子女使用频率低导致内容匮乏 | 中 | APP推送提醒（"3天没给爸爸发消息了"）+ AI自动生成日历节日提醒 |
| PIR误判（宠物等非人体活动） | 低 | PIR+雷达双传感器验证（高级版），基础版配合时间段过滤 |
