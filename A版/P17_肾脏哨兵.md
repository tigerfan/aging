# P17 肾脏哨兵——尿液微流控慢性肾病居家监测仪

---

## A. 需求分析文档

### 1. 产品概述与定位

**肾脏哨兵**是一款基于微流控芯片技术的居家尿液检测系统，通过一次性微流控试片对晨尿进行快速分析，在10分钟内输出肌酐、尿白蛋白、尿素氮、尿酸等8项肾功能相关指标，配合AI趋势分析识别慢性肾病（CKD）早期进展信号。设备以马桶侧置仪为主要形态，老人如厕后取少量尿液滴入试片即可完成检测，极度降低操作门槛。

**蓝海点**：慢性肾病早期几乎无症状，绝大多数老年患者在发展至肾衰竭前未被检出；传统尿检须去医院；居家微流控肾功能检测设备几乎空白。

### 2. 目标用户画像

| 维度 | 描述 |
|------|------|
| 核心用户 | 60–80岁，患有高血压/糖尿病（CKD最大风险因素），每年应定期查肾功能者 |
| 次要用户 | 长期服用NSAID止痛药或氨基糖苷类抗生素的老年人（肾毒性风险） |
| 使用场景 | 每周检测1次，晨起如厕时取尿，10分钟出结果 |
| 医疗背景 | 肾内科/全科医生随访，配合3–6个月一次的医院肾功能复查 |

### 3. 市场痛点与机会

- 中国CKD患者约1.2亿，知晓率不足13%（大量患者在透析前才发现）
- 现有居家尿检试纸：仅检测蛋白、pH等基础项目，无法量化肌酐等关键指标
- 微流控芯片成本降至5–15元/片，量产后居家检测可行性高
- 早期CKD（G1–G2期）干预可延缓肾功能下降10–20年
- 糖尿病肾病（DKD）是透析最主要病因，早筛早干预市场极大

### 4. 功能性需求（FR）

**FR-01 微流控检测（8项指标）**
- 检测项目：尿肌酐、尿白蛋白、尿白蛋白/肌酐比（UACR）、尿素氮、尿酸、尿蛋白定量、尿糖、尿pH
- 方法：光学比色法（微流控芯片上集成酶促反应试剂）
- 检测时间：10分钟（加样→反应→读数全自动）
- 精度：尿白蛋白CV<10%，肌酐误差<15%（与医院生化仪对比）

**FR-02 微流控试片设计**
- 单次使用一次性试片（30mm×60mm层析卡）
- 样本量：50μL尿液（一滴，取样管配套）
- 试片独立防潮包装（有效期2年），盒装（30片/盒）
- 取样方式：配套取样吸管，或如厕时用专用尿杯接少量

**FR-03 智能读取仪**
- 仪器主体：马桶侧挂式或台面式，插入试片后全自动完成光学读数
- 光学系统：LED光源（3波长）+ CMOS图像传感器，检测比色反应颜色深度
- 试片对准引导：机械导槽+图像识别，防止插入方向错误
- 结果显示：彩色LCD屏+语音播报

**FR-04 趋势分析与预警**
- 历史数据图表：每项指标的趋势折线图（周/月/年维度）
- CKD分期估算：基于eGFR（估算肾小球滤过率）和UACR自动分期
- 进展预警：UACR连续3周上升超20%，或eGFR持续下降，触发"建议就医"提醒
- 月度肾脏健康报告（可导出给肾内科医生）

**FR-05 用药肾毒性监测**
- 用户录入当前用药清单（与用药管家P16联动）
- 系统标注肾毒性药物（NSAID、造影剂、某些抗生素）
- 服用肾毒性药物期间，提示增加检测频率（每3天一次）

### 5. 非功能性需求（NFR）

| 维度 | 指标 |
|------|------|
| 尿白蛋白检测范围 | 1–500mg/L（覆盖微量蛋白尿至大量蛋白尿） |
| 肌酐检测范围 | 0.5–30mmol/L |
| 操作步骤 | ≤3步（取尿→插试片→等结果），单手可操作 |
| 仪器尺寸 | 台面版：120mm×80mm×40mm；侧挂版：160mm×60mm×30mm |
| 结果存储 | 本地存储最近2年检测记录 |
| 试片保存 | 常温（15–30°C）保存，有效期2年 |

### 6. 约束条件与合规要求

- 微流控检测设备须作为**II类体外诊断（IVD）医疗器械**申报
- 试片须通过NMPA批准方可宣传医疗检测功能
- 产品声明：居家检测结果为参考值，不替代医院检验，异常需复查
- 尿液样本在检测完成后立即丢弃，不存储任何生物样本数据

### 2. 硬件设计方案

**2.1 微流控芯片（试片）**
- 基底：PMMA（聚甲基丙烯酸甲酯）
- 微流道：激光切割+热压成型，流道宽200μm，深100μm
- 试剂区：预填充干燥酶底物试剂（肌酐酶法、白蛋白溴甲酚绿法等）
- 反应腔：8个独立反应腔（对应8项检测）
- 样本分配：毛细管力自动分液（无需外部泵）

**2.2 光学读数模块**
- 光源：RGB LED（3波长：450nm/540nm/630nm），分时照射
- 探测：CMOS图像传感器（200万像素），拍摄比色腔颜色
- 比色算法：LAB色彩空间颜色提取 → 标准曲线查值 → 浓度计算
- 恒温控制：PTC加热片维持37°C±0.5°C（酶反应温度）

**2.3 仪器主控**
- MCU：STM32H7（运行图像处理+比色算法）
- 显示：2.4英寸IPS LCD，320×240，字体≥20pt
- 语音：TTS芯片（SYN6658），播报检测结果和提醒
- 通信：WiFi（自动上传结果至APP）+ BLE（配对手机）
- 电池：2,000mAh锂电，USB-C充电，待机1个月

### 3. 核心算法设计

**3.1 比色浓度计算**
```python
def compute_concentration(image_roi, standard_curve):
    # 提取比色腔LAB颜色（L亮度，a红绿轴，b蓝黄轴）
    lab = cv2.cvtColor(image_roi, cv2.COLOR_BGR2LAB)
    a_value = np.mean(lab[:,:,1])  # 红绿轴（多数比色反应相关）
    
    # 标准曲线插值（二次多项式拟合）
    concentration = np.polyval(standard_curve.coefficients, a_value)
    return max(0, concentration)

def compute_uacr(albumin_mg_L, creatinine_mmol_L):
    # UACR（尿白蛋白肌酐比）= 尿白蛋白(mg) / 尿肌酐(g)
    creatinine_g_L = creatinine_mmol_L * 0.113  # 单位换算
    uacr = albumin_mg_L / creatinine_g_L  # mg/g
    return uacr
```

```python
def detect_ckd_progression(uacr_history, trend_weeks=4):
    if len(uacr_history) < trend_weeks:
        return None
    recent = uacr_history[-trend_weeks:]
    slope = linear_regression_slope(recent)
    pct_change = slope * trend_weeks / uacr_history[-trend_weeks] * 100
    
    if pct_change > 30:  # 4周内上升超30%
        return "UACR持续上升，建议本周内就诊肾内科"
    elif recent[-1] > 300:  # 大量蛋白尿（高风险）
        return "尿蛋白明显升高，建议尽快就诊"
    return None
```

### 4. 用户交互设计

**检测流程（老人端，极简3步）**
```
如厕后用取样管吸取约2mL尿液（或用随附尿杯接取）
→ 将尿液滴入试片取样孔（2滴即可）
→ 将试片插入仪器（单方向防反插）
→ 仪器自动运行（10分钟）
→ 语音播报结果："今日肾脏检测完成，尿蛋白正常，尿酸偏高"
→ APP推送详细数据和趋势图
```

**APP界面**
- 仪表盘：8项指标颜色状态（绿/黄/红），一目了然
- 趋势图：点击任意指标查看最近3个月趋势
- 就医建议：异常项自动生成"建议就医时告知医生"的简报
