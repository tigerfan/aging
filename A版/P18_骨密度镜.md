# P18 骨密度镜——超声骨质疏松居家筛查仪

---

## A. 需求分析文档

### 1. 产品概述与定位

**骨密度镜**是一款基于定量超声技术（QUS）的居家骨质疏松筛查设备，通过测量足跟骨或桡骨远端的超声传导速度（SOS）和宽带超声衰减（BUA），估算骨质密度T值，为老年人提供方便快捷的骨质疏松风险评估。结合跌倒风险评估和钙/维生素D摄入记录，为老年人提供个性化骨骼健康管理方案，并在T值达到骨质疏松标准时及时提醒就医行DEXA确诊。

**蓝海点**：骨质疏松患者中仅约20%被诊断，DEXA检查仅在三甲医院；QUS消费级设备技术成熟但面向老年居家市场者几乎空白；骨折预防意识正在快速上升。

### 2. 目标用户画像

| 维度 | 描述 |
|------|------|
| 核心用户 | 60岁以上女性（绝经后骨量流失加速），70岁以上男性 |
| 次要用户 | 长期使用糖皮质激素的老年患者（药物性骨质疏松高风险） |
| 使用场景 | 每月自测一次，配合营养干预和运动监测 |
| 临床背景 | 骨科/风湿科/内分泌科医生随访 |
| 核心关注 | T值趋势、骨折风险评分（FRAX）、营养和运动干预效果 |

### 3. 市场痛点与机会

- 中国骨质疏松患者约9,000万，骨折（尤其髋部骨折）是老年人致残致死的主要原因
- DEXA骨密度检查费用200–500元，需专门去医院，大多数老年人从未检测过骨密度
- 定量超声（QUS）技术无辐射、设备便携、成本低，已在基层医疗广泛使用
- 消费级QUS设备（足跟骨）精度已可达到临床筛查标准
- 骨质疏松药物（双膦酸盐）需长期服用，居家监测治疗效果是强需求

### 4. 功能性需求（FR）

**FR-01 超声骨密度测量**
- 测量部位：足跟骨（胫骨跟骨界面，QUS最成熟部位）
- 测量参数：超声传导速度（SOS，m/s）、宽带超声衰减（BUA，dB/MHz）、量化超声指数（QUI/SI）
- 测量精度：SOS重复性CV<0.3%，QUI精度对比DEXA相关系数r>0.75
- 测量时间：<30秒完成单次测量

**FR-02 T值与骨折风险评估**
- 根据QUI自动计算等效T值（基于大样本人群校准）
- FRAX骨折风险计算器（输入年龄、性别、体重、用药史等）：10年主要骨质疏松性骨折概率
- 分级提示：
  - T值≥-1.0：正常
  - T值-1.0至-2.5：骨量减少，建议生活干预
  - T值<-2.5：骨质疏松，建议就医确诊和药物干预

**FR-03 骨健康干预管理**
- 钙摄入记录：每日饮食钙估算（基于食物记录），与推荐量对比
- 维生素D状态追踪（日晒时间记录+补充剂记录）
- 抗阻运动提醒：每周3次负重运动（楼梯、快走）对骨量保护的重要性
- 个性化建议：钙补充剂推荐、日晒建议、运动处方

**FR-04 治疗效果监测**
- 药物服用记录（双膦酸盐等骨质疏松药物）
- 每季度测量趋势：T值/QUI变化图
- 治疗6个月后对比基线，评估药物效果

**FR-05 就医导引**
- T值<-2.5或FRAX高风险：自动生成就医建议书（含历史测量数据），推荐去DEXA确诊
- 附近骨质疏松门诊导航（对接大众点评/医院地图）

### 5. 非功能性需求（NFR）

| 维度 | 指标 |
|------|------|
| 测量重复性 | 同一人5次重复测量SOS标准差<1.5m/s |
| 操作难度 | 老人独立操作≤3步（坐下→放脚→等结果） |
| 测量时间 | <30秒出结果 |
| 防护等级 | IP22（防脚汗、液体溅入） |
| 耦合介质 | 预置水囊式（无需手动涂耦合剂）或自动喷水 |
| 设备尺寸 | 主机200mm×200mm×80mm（地面放置） |
| 数据存储 | 本地存储最近5年测量记录 |

### 6. 约束条件与合规要求

- QUS骨密度筛查设备作为**II类医疗器械**申报（参考类别：6821体外诊断仪器）
- 产品标注：QUS无法完全等同DEXA，T值<-2.5必须就医行DEXA确诊后才能开始药物治疗
- 孕妇、足部皮肤破损、足部石膏固定者不适用
- 耦合水囊：超声耦合凝胶或水囊须符合医用级卫生标准

## B. 详细设计方案

### 1. 系统架构

```
[骨密度镜主机（足跟夹持式）]
     ├── 超声换能器对（发射+接收，1MHz宽带）
     ├── 水囊耦合系统（自动充水）
     ├── 足跟定位导槽
     ├── 超声信号处理电路
     ├── 2.8英寸彩屏
     └── 主控MCU + WiFi/BLE
          │ WiFi
          ▼
[手机APP]
     ├── T值趋势图
     ├── FRAX计算器
     ├── 钙/维生素D管理
     └── 就医导引
```

### 2. 硬件设计方案

**2.1 超声测量系统**
- 换能器：两个宽带压电陶瓷换能器（PZT-5H，中心频率1MHz，带宽0.2–1.8MHz）
- 水囊耦合：硅胶水囊（充水后紧贴足跟两侧），自动泵水充水（测量时）/放水（测量后）
- 接收电路：低噪声前放（LNA）+ 程控增益放大器（PGA）+ 高速ADC（80MHz采样率）
- 信号处理：DSP计算SOS（发射到接收的渡越时间/跟骨宽度）和BUA（频域分析衰减斜率）

**2.2 主控与显示**
- MCU：STM32H7（运行快速傅里叶变换FFT，计算BUA频谱）
- 显示：2.8英寸IPS彩屏，320×240，显示T值、QUI分数和颜色风险等级
- 语音：TTS播报结果（"您的骨密度T值为-1.8，属于骨量减少"）

**2.3 定位与结构**
- 足跟定位：V形导槽+可调宽度（适应足跟宽3–6cm），弹簧夹持确保换能器贴合
- 主机底座：稳固放置于地面，底部防滑橡胶垫

### 3. 核心算法设计

**3.1 SOS（超声传导速度）计算**
```python
def compute_sos(received_signal, transducer_distance, fs=80e6):
    # 检测超声波到达时间（过阈值法）
    threshold = 0.1 * max(abs(received_signal))
    arrival_idx = np.argmax(abs(received_signal) > threshold)
    transit_time = arrival_idx / fs  # 秒
    sos = transducer_distance / transit_time  # m/s
    return sos

def compute_bua(tx_spectrum, rx_spectrum, freq_range=(0.2e6, 0.6e6)):
    # BUA = 频带内衰减量/频率宽度 (dB/MHz)
    attenuation = 20 * np.log10(tx_spectrum / rx_spectrum)
    slope = linear_regression_slope(freqs, attenuation, freq_range)
    return slope  # dB/MHz
```

**3.2 QUI（量化超声指数）计算**
```
QUI = 0.41 × (SOS + BUA) + 3.705
（Bauer公式，临床验证公式，对比DEXA相关系数r≈0.75）

等效T值计算：
T = (QUI - 正常年轻女性均值QUI) / 正常年轻女性QUI标准差
```

**3.3 FRAX自动填写**
- 系统预填：年龄、性别（注册时填写），T值（最近一次测量）
- 用户补充：身高体重、父母骨折史、吸烟史、激素使用史（一次录入，长期保存）
- 输出：10年内发生髋部骨折概率（%）和主要骨质疏松骨折概率（%）

### 4. 用户交互设计

**测量流程**
```
坐下，将足跟放入设备导槽（语音引导：右脚/左脚）
→ 设备自动充水耦合（约5秒）→ 开始超声测量（约15秒）
→ 屏幕显示QUI分数 + T值 + 颜色风险等级（绿/黄/红）
→ 语音播报："本次骨密度T值为-1.5，骨量轻度减少，建议增加钙摄入和抗阻运动"
→ APP推送详细报告和干预建议
```

### 5. 商业化路径与技术风险

**商业化路径**
- **硬件**：1,200元（主机）
- **服务**：免费基础版（本地存储）/ 99元/年（云端趋势分析+医生报告）
- **渠道**：骨科/内分泌科医院推荐、社区卫生中心、药房渠道

**技术风险评估**

| 风险 | 等级 | 应对措施 |
|------|------|----------|
| QUS与DEXA相关性有限（不同部位） | 高 | 明确定位为"筛查工具"，T值<-2.5强烈建议DEXA确诊 |
| 足跟定位不准确导致重复性差 | 高 | V形导槽精确定位，每次自动拍照记录定位情况（存档比对） |

| 老年人对超声技术认知度低 | 低 | 简单宣传：这和医院B超一样，只是测骨骼，无辐射、无痛 |
