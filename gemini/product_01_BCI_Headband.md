# 产品01：基于脑机接口的认知训练发带

## 1. 产品概述
**愿景**：通过无创脑机接口技术（BCI）与多感官刺激，为老年人提供居家日常的认知功能评估与训练，延缓阿尔茨海默症（AD）等认知障碍的发生。
**目标用户**：60岁以上存在轻度认知障碍（MCI）风险或希望预防认知衰退的老年群体。
**核心痛点**：传统认知筛查需要去医院进行量表测试，耗时且依从性差；现有的认知训练游戏缺乏生理层面的反馈与干预手段，效果难以量化。

## 2. 需求分析
### 2.1 功能需求
1. **脑电波（EEG）实时监测**：采集前额叶等关键区域的脑电信号，分析专注度、疲劳度和放松度。
2. **多感官闭环干预**：结合App提供特定频率（如40Hz）的声光频闪刺激，同步促进大脑节律同步化（Gamma波同步）。
3. **认知功能评估**：内置基于BCI的短时记忆、注意力分配等适老化游戏测试，通过EEG数据进行客观评分。
4. **数据追踪与预警**：生成长期的认知趋势报告，若发现异常衰退迹象，自动向家属或社区医生发送预警。

### 2.2 非功能需求
1. **佩戴舒适性**：设备需轻便（<100g），采用柔性干电极，不压迫头部，透气吸汗。
2. **续航能力**：单次充电可连续使用至少7天（每天使用30分钟）。
3. **易用性**：极简操作，戴上自动开机并连接手机/平板，无多余按键。

### 2.3 软硬件接口需求
1. **无线通信**：蓝牙5.2低功耗传输EEG数据。
2. **充电接口**：磁吸式无线充电或Type-C。

## 3. 详细设计方案
### 3.1 硬件架构
- **主控芯片**：低功耗MCU（如Nordic nRF52840），负责数据预处理与蓝牙传输。
- **传感器模块**：
  - 柔性高分子导电电极（干电极）：布置于Fp1、Fp2（前额叶）及耳垂（参考电极）。
  - 高精度EEG模拟前端（AFE，如ADS1299）：实现低噪音脑电信号放大与模数转换。
  - 6轴IMU：检测头部运动轨迹，过滤运动伪影（Motion Artifact）。
- **反馈执行模块**：内置微型骨传导振子及柔和RGB LED阵列（提供光刺激）。
- **电源管理**：聚合物锂电池（500mAh），带充放电保护电路。

### 3.2 软件系统
- **移动端App**：
  - UI层：大字体、高对比度设计。包含每日训练任务推流、认知游戏界面、家属看板。
  - 业务层：设备连接管理、数据同步、云端模型调用。
- **云端服务**：
  - 数据湖：存储脱敏的长期脑电与测试结果。
  - AI引擎：基于深度学习的EEG特征提取与AD风险预测模型。

### 3.3 算法逻辑
1. **信号预处理**：带通滤波（0.5-50Hz）、陷波滤波（50/60Hz）消除工频干扰；基于ICA（独立成分分析）去除眼电/心电伪影。
2. **特征提取**：计算各频段（Alpha, Beta, Theta, Gamma）的绝对/相对功率谱密度。
3. **状态解码**：通过支持向量机（SVM）或LSTM网络，实时输出用户的认知负荷和专注水平指数（0-100）。
4. **闭环调控**：当专注度下降时，App自动调整认知训练游戏的难度，或改变40Hz声光刺激的强度。

## 4. 商业与落地可行性分析
- **成本预估**：硬件BOM成本约150-200元，主要在AFE芯片与柔性电极。量产零售价可定于999-1500元区间。
- **技术难点**：干电极在有头发或皮肤油脂影响下的接触阻抗问题，以及微弱脑电信号的抗干扰处理。
- **适老化交互设计**：考虑到老人可能抗拒“医疗器械”感，外观设计应类似普通运动发带或针织帽边缘，无缝融入日常穿搭。
