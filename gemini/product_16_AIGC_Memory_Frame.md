# 产品16：基于AIGC的个人回忆录生成智能相框

## 1. 产品概述
**愿景**：利用大语言模型（LLM）与AI生成内容（AIGC）技术，通过日常与老人的自然对话，潜移默化地挖掘、记录并整理老人的生平故事，自动生成图文并茂的家族数字传记。
**目标用户**：希望传承家族记忆、有倾诉欲但不会使用智能手机打字记录的老年人。
**核心痛点**：老年人有丰富的阅历和回忆，但缺乏记录能力和倾听者；传统的口述历史需要专业的采访者耗费大量时间整理；老照片逐渐发黄破损，缺乏数字化的故事绑定。

## 2. 需求分析
### 2.1 功能需求
1. **主动式访谈对话**：相框内的AI虚拟助手（或语音形象）能在日常闲暇时，主动向老人抛出话题（如“爷爷，您年轻时下乡插队遇到过什么有趣的事吗？”），引导老人讲述。
2. **多模态记忆捕捉**：老人可直接将老照片放在相框前，AI通过摄像头扫描并询问照片背后的故事，将语音与图像自动绑定。
3. **AIGC回忆录生成**：将老人的口述录音自动转写、提取关键实体（时间、地点、人物）、润色润色，并结合老照片（使用AI修复上色）或AI生成的插图，排版成精美的数字相册或可打印的传记。
4. **家族云端共享**：生成的传记片段实时同步至家属微信小程序，子孙可以随时查看、点赞和评论，甚至补充回忆。

### 2.2 非功能需求
1. **极简交互**：外观与普通数字相框无异，插电即用，无复杂层级菜单。
2. **情感化倾听**：AI语音需具备高度的同理心，适时给出“嗯”、“太不容易了”、“后来呢”等反馈，鼓励老人多说。
3. **隐私安全**：对话内容需在本地进行脱敏处理后，加密上传至家庭专属私有云空间。

### 2.3 软硬件接口需求
1. **视觉输入**：顶置高清微距摄像头，用于扫描实体老照片。
2. **网络接口**：Wi-Fi连接或内置4G/5G eSIM（方便无宽带的空巢老人使用）。

## 3. 详细设计方案
### 3.1 硬件架构
- **显示与交互**：
  - 10-15英寸防眩光（Anti-Glare）触控屏，采用类纸护眼显示技术。
  - 远场麦克风阵列与高保真扬声器。
- **计算平台**：
  - 基于安卓的智能主板，集成轻量级NPU，支持离线唤醒和本地照片OCR/增强预处理。

### 3.2 软件系统
- **访谈引擎（基于LLM）**：
  - 预设访谈大纲模板（童年、求学、爱情、工作、子女）。
  - 根据老人的回答动态调整追问策略，具备多轮对话记忆能力。
- **AIGC内容生产流水线**：
  - 语音转文本（ASR）：支持多种方言识别。
  - 文本摘要与润色：LLM将口语化、碎片化的叙述重写为富有文学色彩的回忆录章节。
  - 图像增强与生成：调用云端Stable Diffusion或Midjourney API，根据文本描述生成当时的场景插画（当缺乏真实老照片时），或对扫描的老照片进行划痕修复和黑白上色。
- **家属互动小程序**：
  - 支持一键将积累的故事生成PDF格式并下单打印实体书。

### 3.3 商业与落地可行性分析
- **成本预估**：硬件BOM成本约500-800元，主要成本在于大屏幕与摄像头。零售价可定于1200-1999元，并提供增值服务（如高级排版、实体书印刷、云端存储空间扩容）。
- **技术难点**：对老年人重口音或非标准普通话的高准确率识别；大模型在生成回忆录时的“幻觉”控制（必须忠于老人讲述的事实）。
- **适老化交互设计**：屏幕默认轮播已经整理好的图文故事或子女上传的全家福；当识别到老人坐在相框前看着某张老照片发呆时，AI会温柔地主动询问：“这张照片里的人是谁呀？”
