# 产品14：自适应变焦与防抖智能餐桌系统

## 1. 产品概述
**愿景**：从“防抖勺”升级为整体的无障碍进食微环境，结合智能餐具与主动补偿桌面，为帕金森病、中风后遗症、重度手颤老人提供尊严、整洁、自主的就餐体验。
**目标用户**：帕金森病中晚期（静止性震颤/运动迟缓）、小脑共济失调、特发性震颤导致无法使用普通餐具独立进食的老年群体。
**核心痛点**：现有的防抖勺虽然能稳定勺头，但老人常无法将食物从碗中准确舀起，且端碗或夹菜时汤汁四溅；防抖勺本身偏重，老人长时间举持易疲劳。

## 2. 需求分析
### 2.1 功能需求
1. **餐具与桌面协同防抖**：防抖勺/筷解决手部高频震颤，而智能餐垫/托盘通过主动随动机制（如万向平移与倾斜），解决餐具靠近碗碟时的相对位移，让“碗找勺”。
2. **防泼洒自平衡碗盘**：基于内置陀螺仪与微型无刷电机，碗盘在端起或移动过程中，即使手部大幅晃动，内部汤汁也能保持绝对水平线。
3. **自适应视距变焦餐桌灯**：顶部集成AI摄像头与液态透镜/变焦光源，根据老人头部的前倾或后仰，自动调节桌面照明的光斑大小与亮度，并在特定菜肴上打出高亮聚光（帮助视力衰退者看清鱼刺或骨头）。
4. **进食动作与营养监测**：通过勺子传感器和桌面称重传感器，精准记录每口食物的摄入量、频率及总重量，评估是否有吞咽困难（如进食停顿过长）。

### 2.2 非功能需求
1. **全身防水与易清洗**：所有智能餐具及餐垫必须达到IPX7级防水，支持水洗甚至洗碗机清洗（可拆卸电子模块）。
2. **食品级材质与防烫**：接触食物部分采用医用级316L不锈钢或Tritan材质，碗壁需双层真空隔热。
3. **轻量化**：防抖勺重量需控制在80g以内，降低手腕负担。

### 2.3 软硬件接口需求
1. **多设备互联**：勺、碗、餐垫之间通过低功耗蓝牙（BLE）或私有2.4G协议组网，实现毫秒级数据同步。
2. **统一充电**：智能餐垫本身也是一个大面积的无线充电板，餐具放上即可充电。

## 3. 详细设计方案
### 3.1 硬件架构
- **智能防抖勺/筷**：
  - 微型三轴/两轴无刷电机云台（隐藏于手柄内）。
  - 高频IMU（捕捉手部震颤频率与幅度）。
- **主动补偿餐垫/托盘**：
  - 磁悬浮直线电机或多轴步进电机驱动的平移/倾斜台（藏于防水垫片下）。
  - 高精度应变片（称重传感器）。
- **自平衡碗**：
  - 碗底集成微型万向节与姿态传感器，外壳与内胆分离，内胆随手部晃动反向补偿。
- **视觉照明中控**：
  - AI视觉芯片（如K210），控制顶部的LED阵列或变焦马达。

### 3.2 软件系统
- **协同抗震算法**：融合勺端高频震颤信号与碗端低频运动信号，餐垫通过预测勺尖的运动轨迹，主动微调碗的倾角和位置，降低舀取食物的难度。
- **视觉高光追踪算法**：识别桌面上不同菜品的边缘，当老人的手（或防抖勺）靠近某盘菜时，顶灯自动增加该区域的照度和显色指数（CRI>95），让红肉更红、绿菜更绿，刺激食欲。
- **营养分析系统**：根据餐前餐后的重量差，结合App中预先录入的菜谱，估算卡路里与宏量营养素摄入。

### 3.3 商业与落地可行性分析
- **成本预估**：包含勺、碗、餐垫及顶灯的整套系统BOM约1500-2000元，作为高端康复辅具，零售价可达4000-6000元。
- **技术难点**：轻量化微型电机的扭矩与体积的矛盾；不同震颤频率（帕金森4-6Hz，特发性6-12Hz）的自适应滤波算法。
- **适老化交互设计**：完全无需开关机操作。餐具拿起自动启动防抖，放下自动进入休眠；清洗时只需拔下磁吸手柄尾部的电池仓。
