# 产品23：吞咽监测与防噎智能项圈

## 1. 产品概述
**愿景**：针对老年人因吞咽肌肉退化导致的高频次呛咳和异物卡喉致死风险，通过贴合喉部的微型多模态传感器，实现进食过程中的实时吞咽质量评估与致命性窒息的瞬间报警。
**目标用户**：患有脑卒中后遗症、帕金森病、阿尔茨海默症晚期，或因衰老出现严重吞咽困难（Dysphagia）、极易发生误吸或噎食的老年人。
**核心痛点**：老年人异物卡喉（如汤圆、果冻、大块肉类）发生时往往无法发声呼救，短短几分钟便会因缺氧脑死亡；日常喂食中，护工难以准确判断老人是否已将上一口食物完全咽下，盲目连续喂食极易引发吸入性肺炎。

## 2. 需求分析
### 2.1 功能需求
1. **吞咽动作识别与分期**：精确捕捉喉结上抬、甲状软骨运动等特征，判断当前处于口腔期、咽期还是食管期，评估吞咽是否彻底完成。
2. **误吸与呛咳预警**：通过分析喉部高频声学信号（如微弱的咳嗽声、气流呼噜声或“湿性嗓音”），发现食物或液体误入气管的迹象，立即通过震动提醒老人或家属停止进食。
3. **完全性气道梗阻（噎食）极速报警**：一旦监测到吞咽动作后，呼吸气流声完全消失且伴随剧烈的颈部肌肉痉挛/挣扎电信号，系统判定为气道梗阻，在5秒内发出最高级别的高分贝声光警报，并自动拨打急救电话。
4. **进食节律引导**：内置微型振子，根据老人的吞咽能力，提供合适的进食节奏（如“震动一次代表可以喂下一口”），防止进食过快。

### 2.2 非功能需求
1. **无压迫佩戴**：采用极轻的柔性硅胶或记忆钛合金颈环设计（类似细项链），对气管和颈动脉无任何压迫感。
2. **防误报机制**：能准确区分正常的清嗓子、说话、深呼吸与真实的噎食。
3. **防水防油污**：进食场景极易沾染汤汁，设备需达到IPX7级防水，支持直接水洗。

### 2.3 软硬件接口需求
1. **无线通信**：蓝牙低功耗（BLE）连接手机App或养老院中控台。
2. **充电接口**：磁吸式触点充电，防盲插。

## 3. 详细设计方案
### 3.1 硬件架构
- **多模态感知模块（紧贴甲状软骨区域）**：
  - 压电薄膜传感器（Piezoelectric Sensor）：捕捉喉部微小的机械震动和喉结位移轨迹。
  - 接触式高频麦克风（类似于听诊器）：拾取吞咽声、呼吸音和咳嗽声。
  - 表面肌电（sEMG）干电极：监测舌骨上肌群的收缩力度。
- **主控与报警模块（位于颈后或锁骨下方）**：
  - 超低功耗MCU（集成轻量级DSP）。
  - 高分贝蜂鸣器（>90dB）与震动马达。
  - 电池仓（100mAh左右，因仅在进食期间使用，可续航数周）。

### 3.2 软件系统
- **吞咽声机电联合特征提取**：
  - 单一传感器极易受干扰（如咀嚼声）。算法需融合机械位移（证明有吞咽动作）、肌电信号（证明有发力）和声学信号（判断是否顺畅），通过CNN或SVM模型输出“安全吞咽”、“疑似误吸”或“气道完全梗阻”的状态。
- **自适应基线校准**：
  - 不同老人的皮下脂肪厚度和肌肉力量差异巨大。设备在初次使用的前几口进食中，自动学习并建立该老人的正常吞咽基准模型。
- **家属/医生看板**：
  - 记录每日的吞咽次数、平均耗时、误吸频率，生成吞咽功能衰退趋势图，为医生调整饮食黏稠度（如是否需要添加增稠剂）提供客观依据。

### 3.3 商业与落地可行性分析
- **成本预估**：硬件BOM（高精度压电与麦克风组合）约150-250元。零售价预估599-899元。
- **商业模式**：除了2C销售，更适合作为养老机构防范护理风险（噎食致死是养老院最大的赔偿黑洞之一）的标配辅具。
- **技术难点**：在颈部复杂的肌肉运动（如转头、说话）中提取纯净的吞咽特征；算法在极端危急情况下的“零漏报”与日常使用中的“低误报”之间的平衡。
- **适老化交互设计**：护工在开饭前给老人戴上（磁扣一吸即开机），饭后取下。一旦发生严重卡喉，项圈不仅狂响，还会亮起刺眼的红光，引导旁人立即实施海姆立克急救法。
