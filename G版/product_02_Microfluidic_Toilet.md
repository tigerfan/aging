# 产品02：微流控血液/尿液即时分析智能马桶

## 1. 产品概述
**愿景**：实现家庭环境下的无感、高频次生理指标监测，将日常排泄行为转化为医疗级健康数据的持续追踪入口。
**目标用户**：患有糖尿病、肾病、痛风等慢性疾病，需频繁监测生化指标的老年人。
**核心痛点**：传统采血/采尿测试需侵入性操作或繁琐的手工取样，老年人依从性低，且难以形成高频连续的数据曲线以指导用药和饮食。

## 2. 需求分析
### 2.1 功能需求
1. **全自动样本采集**：在用户如厕过程中，通过隐蔽的机械臂或引流槽自动截取中段尿或检测微量便血。
2. **微流控即时分析（POCT）**：利用一次性/可替换微流控芯片，检测尿糖、尿蛋白、尿酸、酮体、红细胞等核心指标。
3. **数据清洗与上云**：将检测结果与基线数据对比，剔除异常值，加密上传至云端健康档案。
4. **自清洁与耗材管理**：每次检测后自动进行高温/高压水流冲洗消毒；自动提醒芯片盒或试剂耗材余量。

### 2.2 非功能需求
1. **无感交互**：用户无需改变任何原有如厕习惯。
2. **卫生与防交叉感染**：取样模块与分析模块物理隔离，确保无细菌滋生。
3. **适老化反馈**：马桶侧边配备大屏/语音播报，用“红黄绿”灯直观显示当日指标健康度。

### 2.3 软硬件接口需求
1. **水路接口**：标准马桶进水/排水管线对接。
2. **供电接口**：标准220V交流电插座。
3. **网络接口**：Wi-Fi模块，支持一键配网。

## 3. 详细设计方案
### 3.1 硬件架构
- **核心模块**：
  - 取样流道：采用仿生疏水材料涂层，防止残留。
  - 微型蠕动泵与阀门系统：精确控制样本与试剂的混合比例。
  - 光学/电化学检测舱：集成多波长LED光源与光电二极管（吸光度法），或高灵敏度电化学传感器（电流法）。
- **外围设备**：
  - 智能马桶盖底座：集成体重秤与体脂分析电极（通过大腿后侧接触面）。
  - 耗材仓：可容纳30天的多参数测试卷带或芯片阵列。

### 3.2 软件系统
- **设备端OS**：RTOS（如FreeRTOS），保证微流控时序控制的实时性与可靠性。
- **用户端App/小程序**：
  - 可视化图表展示每日/周/月指标波动趋势。
  - 饮食与用药打卡记录，与生理指标关联分析。
- **云端服务**：
  - 异常值报警系统：当连续出现严重超标（如突发性蛋白尿）时，向监护人及社区签约医生发送告警。

### 3.3 算法逻辑
1. **身份识别**：通过体重、如厕习惯时间甚至马桶圈接触面的生物阻抗（BIA）特征识别当前家庭成员（区分老爷爷和老奶奶）。
2. **浓度校准算法**：根据尿液的比重/电导率校准检测物的绝对浓度，消除因饮水过多导致的稀释误差。
3. **预测模型**：利用时间序列分析（如ARIMA模型）预测特定指标（如尿酸）的临界点，提前建议干预。

## 4. 商业与落地可行性分析
- **成本预估**：硬件BOM成本约3000-5000元。商业模式可采用“硬件低利润/平价+耗材（芯片/试剂）持续收费（如每月100元）”的订阅制。
- **技术难点**：长期使用下的管路结垢防堵塞技术；尿液复杂成分对光学/电化学检测的干扰。
- **适老化交互设计**：考虑到老人可能不擅长看手机，直接在马桶水箱盖上投射清晰的大字报告（如“今日血糖平稳，请继续保持”）。
